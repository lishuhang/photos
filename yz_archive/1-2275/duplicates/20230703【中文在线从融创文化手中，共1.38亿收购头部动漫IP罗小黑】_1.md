# 中文在线从融创文化手中，共1.38亿收购头部动漫IP罗小黑

6月28日，中文在线（300364）发布公告称，公司拟以1.38亿元人民币收购北京寒木春华动画技术有限公司（以下简称“寒木春华”）51.04%股权。这其中有1.03亿元对价收购了梦之城所持38.17%的股权。众所周知，梦之城是融创文化的子公司。另一对价方圣晓文化也是融创文化系的公司。也就是说，罗小黑的IP从融创文化成功的转移到了中文在线手中。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7tMuAmiadsyvAk4l30TZvmgP03RGX0iaosuL5yVawsdblYqeWUcOTHYoQ/640?wx_fmt=png)**头部动漫IP罗小黑的进击之路**

寒木春华创始人章平（即导演木头）于 2010 年创作出“罗小黑”形象，其原型是导演曾养过的一只猫，次年，以“罗小黑”为主角的动画剧集《罗小黑战记》第一季在B站正式上线。《罗小黑战记》讲述的是猫妖小黑受重伤流落街头时被少女罗小白收养，起名为“罗小黑”，从此一起踏入神仙与妖怪的世界中的故事。

尽管每集只有五分钟、更新间隔漫长，《罗小黑战记》依然凭借自身的高质量网罗住了许多观众的心。在做《罗小黑战记》第一季时，因为缺少人手一年只能制作两集，直到渐渐有了名气、团队开始壮大，更新速度才得以加快。

《罗小黑战记》第一季最终在2021年正式完结，收获全网播放量超10亿次，B站约15万人评分9.9分，豆瓣超10万人评分9.6分。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/CkgDGce9CLlQoic7OXG1GZiawaHsIkp0nrgMPlN1KPoa7yDbyuHP3WMUOdLtMmDR16mB0b3ITOK1a3ErcrmibcBbA/640?wx_fmt=png)

2015年，寒木春华开始动画电影《罗小黑战记》的制作。

2019年8月，融创文化集团宣布了对国内动漫IP孵化及商业化运营公司梦之城文化的独家投资，实现对梦之城文化的战略控股，拥有了阿狸、小鹿杏仁儿、罗小黑、皮揣子等多个动漫形象。同年9月，《罗小黑战记》电影顺利上映，取得超过 3 亿元的票房收入，豆瓣超50万人评分8.0分，并入围法国昂西国际动画节的官方长片Contrechamp竞赛单元。

根据国家电影局网站的备案公示信息，《罗小黑战记 2》已进行备案公示，预计将在2024年底制作完成，系列化的IP大电影将持续为“罗小黑”IP价值放大增值。

成立于 2000 年的中文在线是国内最大的正版数字内容提供商之一，旗下拥有 17K 小说网四月天小说网、汤圆创作、万丈书城多个原创网络平台，同时积极挖掘有声、微短剧、动漫及影视的IP 价值，并布局动漫、影视、海外等业务。

本次战略并购寒木春华，是中文在线2021年提出的“决胜 IP”的举措之一，1.38 亿的收购价格证明了“罗小黑”作为优秀IP 具有巨大价值和市场潜力。

其中，公司以人民币10307.5321万元的对价购买北京梦之城文化有限公司所持目标公司38.1760%的股权，以人民币3473.6450万元对价购买圣晓文化娱乐（北京）有限公司所持目标公司12.8654%的股权。也就是说，寒木春华估值约为2.7亿元。

截止到2023年3月31日，寒木春华未经审计的净资产为3048万元，其2022年的净利润为955万元。

中文在线不是第一次在动漫领域有所动作。此前，由中文在线出品的动态漫《混沌剑神》已在爱优腾三平台上线，中文在线与企鹅影视联合出品、原力动画制作的动画《修罗武神》也将在腾讯视频独家播出，另有多部精品IP改编的动漫作品处于创作阶段，未来将有机会与观众正式见面。

目前，以“罗小黑”系列IP为核心，寒木春华实现了漫画、番剧、动画电影、衍生品的多阶段跨越。“罗小黑”IP形象授权业务也在快速增长，合作品牌涵盖服饰、游戏、饮料、卡片、香水、主题壁纸等多种领域。

公告显示，双方将充分利用各自资源优势，进一步叠加和放大 IP 价值。一方面，“罗小黑”的 IP 价值将在产能效率、IP 衍生品商业化、出海等方面得到进一步放大；另一方面，借助寒木春华的优质 IP 和产能，将进一步提升中文在线海量IP动漫产品化的能力，提高市场竞争力和知名度。
