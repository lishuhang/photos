# 黄晓明点破行业真相：AI越厉害，越离不开IP

“刚才在台下跟同行聊天，大家都很焦虑，不知道未来该怎么样了。”

昨晚，2025阅文IP盛典的舞台上，黄晓明在领取“年度IP合伙人”荣誉时，没有按照惯例单纯致谢，而是说了一段看似“鸡汤”却极其应景的话，意外戳中了当下影视行业所有人的心事。

![图片](http://mmbiz.qpic.cn/sz_mmbiz_jpg/7PwOwGuzgVAXaxXTIKWFKIibbax9GHnoCOlp8WT9FMJDKYf7QbgzicoYh7U0CZCPUNntQUicicl6LIvclR02eQNWzdmqykSwrwwJvAKFCmsObyY/640?wx_fmt=jpeg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

黄晓明在2025阅文IP盛典领奖

“其实我后来想了一下，AI是什么，AI就是爱啊。所以我们要爱好的故事、IP和作者，要好好地去演绎，爱所有的观众。只要我们用心地去做项目、创造IP，那自然就会有好的观众来爱我们。”

这段话迅速冲上热搜。大家记住了他顺便宣传的迪丽热巴新剧《白日提灯》，却很少有人留意，这短短几句话背后藏着影视行业的集体焦虑。

行业共识：AI能干活，但造不出IP的灵魂

这种焦虑不是情绪，而是现实。

小娱在之前的报道中曾提到，今年长剧开机率相比巅峰期锐减约90%。若往年开机1000部，今年仅剩100-150部左右。真人短剧同样是如此，头部短剧演员的片酬下降了30%-40%，一些演员长时间无戏可拍。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

《逐玉》海报（图源：豆瓣）

焦虑的另一面，是冰火两重天的现实。《逐玉》等传统的古偶题材依然成为全民话题，两位主演疯狂吸粉数百万，配角也“全员上桌”。动画领域，《斗破苍穹》《凡人修仙传》等年番热度不减，衍生品销量也屡破纪录。这些逆势而上的内容，都有一个共同的底色——它们都源自头部IP。

黄晓明的那句“AI就是爱”，或许就是一种返璞归真：技术越先进，人心越需要好故事。

今年2月底，光线传媒董事长王长田在内部信里直言，AI时代，IP的价值并没有消失，反而被逐渐放大了，而影视行业要坚定地去做AI做不了的事。原创IP背后的创意、审美、人性共情、价值判断和天马行空的想象力都是AI无法复制的。

易凯资本首席CEO王冉的观点极为类似。他认为AI在相当长一段时间里做不了的，就是责任和信任。而IP恰恰是承载这份责任和信任最好的武器。在整个影视行业，IP以及IP的正典权利，是最有价值的长期资产。未来资本会向影视行业流入，但只会流向IP所在的地方。

更有意思的是，现在所有经营状况良好，且拥有优质作品的影视公司，都开始把目光瞄准了IP价值。柠萌影视在最新的财报中向外界透露出的一个重要信号是，强原创和大IP给了他们不断投入影视市场，开发系列化内容的底气。

从业者们逐渐明白：过去拼的是流量与明星，现在拼的是IP这一最硬的通货、最稳的护城河。

而在IP这件事上，阅文几乎站在了行业最中心的位置。

一方面，它起家于网络文学，手握海量IP源头，并已延伸至动漫、剧集、衍生品等全产业链。另一方面，它在AI技术上的积累如“作家助手”、“漫剧助手”——正在为创作者赋能。

过去一年，娱乐资本论观察了市面上主流的影视内容。其中，无论是电影电视剧，还是时下最流行的AI漫剧，几乎都离不开知名IP的改编。每一个大热IP的背后，都拥有着产出不同形式影视作品的改编空间。

阅文在IP储备层面是影视行业的第一梯队。根据云合数据，2025年上新长剧累计有效播放霸屏榜前10位中，有5部改编自阅文IP；全网动漫播放霸屏榜TOP10中，高达9部改编自阅文IP。

阅文CEO兼总裁侯晓楠在今天的阅文IP盛典产业论坛上公开了一组数据：“过去半年，阅文在AIGC视频领域做了一些探索。我们把超过1000部网文转化为了漫剧，其中超100部播放量破千万，26部破亿，自制漫剧成功率达到行业均值的5倍以上。”

其中一个数据至关重要，那就是自制漫剧成功率达到行业均值的5倍以上。众所周知，阅文自制的AI漫剧基本都改编自阅文平台的网文IP。IP所带来的跨圈层流量和内容价值，都在这些AI漫剧身上有所体现。更为重要的是，阅文IP当前的开发率，不足千分之一，99%的宝藏IP还处在休眠之中。如果把这些IP利用起来，成功将非常惊人。

按照过往的经验，中国影视市场能够产出的作品类型仅占全球三成左右。AI技术的应用可以刺激不同类型作品的开发，填补市场空白。受限于过去的成功经验，传统投资模式带来影视内容成本的水涨船高。在AI技术的加持下，影视人可以尝试更多高概念和创新的IP内容，形成新的增量市场。

2025阅文IP盛典上，阅文首次设置了“最具突破IP漫剧”的奖项。获奖者《惊悚乐园》是由阅文IP改编的B站独播漫剧。首集上线48小时之内播放量就突破百万，成功登上B站全站热搜第二名和热门第四名。

惊悚乐园

事实上，行业内也一直在期待，阅文手中的国民级IP何时被改编成AI漫剧。

阅文漫剧业务部总经理杨沾与娱乐资本论交流时也表示，阅文已经在为头部IP长线规划AI漫剧等形式的内容。阅文对于这些头部IP的AIGC内容都有着严格的品质把关。一旦符合阅文内部的质量预期，上述作品就将推向市场。杨沾预计，一些头部IP的AIGC内容有望在今年上线。

“火种计划”登场：让普通人成为“超级个体”

IP 为王已经是行业共识，但问题紧跟着来了：IP宝藏虽好，但如何加速挖掘？普通创作者、小团队，怎么才能分到这波红利？

阅文给出的答案是“火种计划”。

产业论坛上，侯晓楠对外公开了全新的AIGC视频扶持计划，推出开放IP、平台扶持、系统培训、资金支持四大举措。

举措一是开放IP，支持2D、3D、拟真人等多元形式的AIGC视频改编与联合制作。

正如前文所述，阅文的手中握有IP富矿，他们有能力也有动力将这些沉睡的资产唤醒并加以利用。杨沾也曾在采访时表示：“两年之前，我可能想不到阅文在不到几个月的时间，可以把上千个IP改编成AI漫剧。现在，我们反倒觉得这个数字还不够。”

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

产业论坛现场公布火种计划

举措二是平台扶持。通过漫剧大赛，挖掘AI创作者，计划签约1000位AI方向的导演。漫剧大赛已于28日正式启动。针对优秀作品，阅文集团还将为创作者提供100份制作合约，总价值超千万元。

举措三是结合名校资源和实训体验的AI创作者培训。

举措四是资金扶持。阅文将投入1亿元人民币，推出首个个人AI创作者扶持计划。该扶持举措主要面向个人创作者OPC和小型工作室。

除了IP开放，另外三个举措几乎都旨在扶持个人创作者。AI技术的进步给创作者提供了无限的可能。在今天这个时代，只要你拥有好IP和便捷的AI工具，每一个优秀的创作者都能化身“一人公司”或“超级个体”。阅文手中不缺IP，那么他们需要挖掘的正是这种“超级个体”，培育全新的创作者生态。

阅文内部其实有很多优秀的作者，他们拥有转型漫剧导演的能力和构想。Z世代作家已经成为阅文平台的主力军。2025年阅文新签约作家中95后占比高达65%，今年获得年度新锐作家奖项的三人也全部是95后。

现在，阅文已经在鼓励更多网文作家跨界漫剧创作，成为OPC模式的先行者。其中包括了10多位白金大神作家。创作者显然不是单纯的版权出让方，而是IP改编漫剧开发过程中的核心参与者，对IP的改编方向、内容呈现等拥有话语权，确保改编作品尊重原著精神。

跳出公司视角，站到整个行业的高度上看，“火种计划”凸显的是影视行业的竞争壁垒正在回归至IP与新的创作者生态。在“一人时代”，AI让“超级创作者”成为可能，阅文所做的，正是培养更多IP生态中的“超级个体”。

黄晓明用“爱”来解读AI，或许并非只是一种修辞。在这个技术狂奔的时代，真正能够穿越周期的，从来不是炫目的特效，而是那些能与人产生情感共鸣的故事与角色。当AI解决了“如何做”的效率问题，IP的价值，就在于回答那个永恒的命题——“为什么要做”。

一个全新的内容生态图景，正向我们徐徐展开。
