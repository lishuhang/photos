## 步骤 1　创建资产-风格①手绘国漫风

**操作**：点击画布顶部工具栏的「添加节点」按钮 → 在弹出菜单中 hover「创建资产」→ 点击「风格」。画布出现"风格1"节点。

**双击节点标题"风格1"**，重命名为「手绘国漫风」，按 Enter。

**点击节点中央"请输入风格描述..."区域**，粘贴以下提示词，点击画布空白处保存：

```
90年代中国手绘动画风格，参考《大头儿子小头爸爸》《海尔兄弟》《宝莲灯》《哪吒传奇》的线条质感。人物轮廓线为深褐色手绘勾线，平涂色块为主，无日漫赛璐璐渐变、无大眼尖下巴、无小嘴高光；面部以简练的几笔勾出眉眼鼻嘴，保留国漫的拙朴感；背景为手绘水彩晕染，整体色调偏暖黄。禁止：日漫笔触、赛璐璐渐变、真人实拍、3D渲染、写实皮肤毛孔、大眼睛尖下巴。
```

**开启"仅使用风格提示词"开关**（默认可能已开启，确认绿色为开）。

**点击描述区**，粘贴：


> 说明：`@资产名` 引用画布上对应的资产节点。引用方式为：在文本节点的描述中写 `@` + 资产节点名，系统会自动识别。或者在创建文本节点后，从各资产节点的输出点拖线连到文本节点的输入点（按连入顺序自动编号为 @参考图1、@参考图2...）。

**连线**：
1. 从「风格·手绘国漫风」输出点拖线到「sc01-描述词」输入点
2. 从「场景·长安夜街」输出点拖线到「sc01-描述词」输入点
3. 从「场景·厅事」输出点拖线到「sc01-描述词」输入点
4. 从「角色·口技者」输出点拖线到「sc01-描述词」输入点
5. 从「道具·桌椅扇抚尺」输出点拖线到「sc01-描述词」输入点

**生成图片**：点击「sc01-描述词」节点上的「生成」按钮（或右键选"生成图片"），系统用连入的风格+场景+角色+道具+文本生成首帧图。生成后画布自动出现"图片1"节点（sc01 的首帧）。
