# 长视频的下一个十年：长期主义、工业化2.0

曙光、希望、突破、未来……在刚刚结束的第十届网络视听大会上，这几个词成为从业者在演讲及论坛发言时的高频词，影视行业的复苏迹象愈发明显。

在这个过程中，我们可以清晰地感觉到，对长视频平台来说，当下正是具有里程碑意义的关键节点。

一方面，经过过去十年的探索，各平台的财务状况有了一定的改善。爱奇艺已经连续四个季度扭亏为盈，相较2021年1月，去年年底的会员观看时长增长了66%；腾讯视频在去年7月实现首次盈利，目前也接近了盈亏平衡点。

另一方面，在降本增效、回归理性的主张之下，长视频平台在内容生产方面也体现出品质的变化。去年下半年至今，剧集和综艺市场都有不少让观众感到惊喜的项目：《狂飙》《风吹半夏》《三体》《爱情而已》《种地吧》《半熟恋人2》《无限超越班》……

前一个十年已经尘埃落定，在交出业绩与内容成绩单的同时，行业新的机遇和挑战也一并来临，即AIGC的出现。这个词同样高频地出现在不同主题的演讲和论坛上，动画从业者坦言作为创作者因其引发的焦虑，编剧们也提到了自己被ChatGPT取代的可能性，而站在平台的角度上，AIGC将带来的变化很可能是变革性的。

无论是如何保持稳定发展，还是如何再跃上一级台阶，面对下一个十年时，长视频平台的解法是什么？

听完从业者们在网络视听大会上的发言与讨论后，娱乐资本论捕捉到两个关键词，其一是长期主义，其二是影视工业化。

对于国内长视频平台来说，长期主义并不是一个新鲜的主张与策略，但在多年探索之下，不同平台的打法也各有侧重。

以腾讯视频为例，腾讯在线视频首席执行官孙忠怀在演讲中表示，“短期来看，我们将改变过于集中优势品类的策略，全力拥抱多元化的选题和表达，同时我们会以长期主义的思维持续长线IP内容的开发。”

由此可见，腾讯视频将长期主义的落脚点放在了长线IP内容的开发上。

所谓长线IP，即可以开发5-10年的IP储备。这类内容的开发，通常意味着更大的投入、更长的周期、更多的不确定性。但从长远来看，优势也是显而易见的。比如，沉淀出一批粘性更高、更有追随意愿的用户，在品牌效应的加持下，也更有利于释放商业化方面的潜力。

过去提起腾讯视频的长线IP，很多观众的第一反应一定是《鬼吹灯》系列，随着《三体》播出，《三体·黑暗森林》《三体·大史》的官宣，其长线IP又有了一个典型案例。值得注意的是，除了剧集领域，腾讯视频在动漫、纪录片赛道也不乏相关尝试，比如《斗罗大陆》《风味人间》，都是已经取得一定成绩的长线IP。

长期主义同样是优酷在内容上的坚持。阿里文娱总裁兼优酷总裁樊路远提到，为用户长期稳定推出好内容的愿景，平台将“为好内容全力以赴”写进了品牌主张。基于此，优酷选择从源头处出发，将人才培养作为长期主义的具体切入点。

在此之前，优酷就举办过“新号角电视剧导演之夜”，发布了青年导演扶持计划，这回的培养对象则是编剧。在网络视听大会现场，优酷与成都文旅集团联合打造的“中国故事村”项目正式启动，第一个推出的项目是“拔萃·编剧计划”。

据优酷副总裁谢颖介绍，该计划分为短训班和大师班。前者解决从梦想到入门的问题，面向全社会招生，由张巍、何亮、苏晓苑、张勇等知名编剧对入选学员进行技能培训，优酷评估中心、运营中心、剧集制片中台的专家也会提供平台的数据和经验。

后者则解决从入门到出师的问题，短训班表现出色的学员，会在资深编剧的带领下参与优酷剧集项目的制作，经过一到两个项目的锻炼后，能够独立完成内容达标的剧集大纲创作，即可成为优酷的签约编剧。

谢颖同时表示，“拔萃·编剧计划”的周期是五年，在此期间，优酷每年都会在成都举办两次短训班。而这样的周期和频率，同样是奉行长期主义的结果。

内容生产离不开技术支持，影视工业化同样是行业里的老生常谈，这个话题之所以仍有讨论空间，是因为在对科技的追求与运用下，国产影视工业化进程一直在加速发展，从策划、开发、制作、审查到播出，智能生产越来越多地覆盖到影视制作的流程。

而在这个过程中，无论是探索还是应用，长视频平台都扮演了重要角色。在此次的网络视听大会上，我们也能看到各平台对目前工业化成果的分享以及对未来趋势的判断。

爱奇艺创始人、首席执行官龚宇总结，“基于互联网的人工智能、大数据和云计算三大技术，我们搭建的影视工业化的支撑系统，进行商业指标预测分析、影视制作流程规范管理和辅助从业者提高效率。借助科技创新，特别是智能制作，推动影视行业从决策、管理到制作、分发全流程转型升级。”

比如剧本分析工具，当剧本录入系统后，几分钟内就能看到对其段落、结构、人物关系、不同场次占比等方面的分析；影视制片工具则能让制作团队在开拍、录入物料的同时，迅速掌握进度，实时看到对已拍内容的反馈。

具体到项目上的技术运用也有典型案例，去年，爱奇艺和制作公司恒星引力在横店搭建了一座2400平米的虚拟摄影棚，并在那里完成了业界首部运用“XR虚拟制片”的剧集《狐妖小红娘·月红篇》。

优酷也一直在推动影视工业化的发展，面向全行业推出虚拟拍摄、项目管理和影视资产管理等全流程服务。

比如项目管理环节的云尚制片系统，在这个系统中，制片人可以直接管理进度和预算，导演可以在片场快速查询、回看刚刚拍好的素材，线上审片既避免了异地文件传输的难题，也保障了素材的安全。

此外还有连接七大仓库、33万件资产的影视资产数字管理平台，鼓励行业对服装道具进行循环利用，《满江红》、《流浪地球2》等影片的不少道具都出自于此。平台相关负责人曾在接受采访时表示，2019年上线后的两年里，该线上平台服务了300多个剧组，为行业省下过4000万。

除了已有成果的分享，让全行业达成共识的是，技术与内容密不可分的关系以及影视工业化的重要意义。更有从业者提出，现阶段，应该迎来新的影视工业化2.0时代，AIGC的出现也加速了这一节点的到来。

那么至关重要的是，如何看待这一新事物及其带来的行业影响？具体到行业实践中，它又将具体扮演什么样的角色？

对于行业中不同的细分品类或工种来说，这一问题的答案也有所区别。在腾讯视频举办的“以科技驱动艺术拓新边界”主题论坛上，作为深受技术影响的动漫行业从业者，几位嘉宾针对这一话题表达了自己的想法和感受。

在腾讯在线视频平台运营部燃动宇宙工作室群负责人周望看来，尽管需要时间流程化、工业化和模块化，但是在剧本统筹、前期的美术发展以及导演对整体氛围的选择上，AIGC都可以发挥非常大的作用。

接下来，平台对待动画领域技术探索的策略上，会结合三维和二维以及对AIGC的研究突破传统的界限，从而进一步赋能行业。另外，在视觉发展上也可以有更多的题材和想象，在相对的时间里降低成本、提高效率。

站在创作者的视角上，焦虑感在所难免，但更值得关注的是新技术带来的新可能。正如玄机科技董事长沈乐平所言，“这种浪潮过来的时候基本上是无法抗拒的。我们必须要学会新的浪潮下如何畅游，要提升自己的认知和自己的技能。我们来把握新的时代，把握新的技术浪潮。把它转化成非常好的朋友，而不是我们的对手。”
