**作者|吴晓宇**

由中国玩具和婴童用品协会（中国玩协）主办的CTE中国玩具展、CLE中国授权展、CKE中国婴童用品展、CPE中国幼教展于10月16-18日在上海新国际博览中心举行。预言家游报作为合作媒体，与多位行业大佬进行对话，探讨IP授权领域的新变化和新趋势。

阅文集团成立于2015年3月，是一家以数字阅读为基础，IP培育与开发为核心的综合性文化产业集团。阅文旗下囊括QQ阅读、起点读书、新丽传媒、腾讯动漫等业界知名品牌，平台汇聚了强大的创作者阵营、丰富的作品储备，覆盖200多种内容品类，触达数亿用户，已成功输出《庆余年》《赘婿》《鬼吹灯》《全职高手》《斗罗大陆》《琅琊榜》等网文IP及多种形式的改编作品，覆盖有声、动漫、影视、游戏、商品化等多种业务形态。

娱乐资本论·预言家游报对话了阅文集团消费品授权负责人乐敏。乐敏认为毛绒品类可能是IP授权领域下一个风口。搪胶毛绒这样新工艺的出现，对于IP衍生品领域有着重要的意义。同时，2024年对于二次元赛道只是一个开始，未来还有很多潜力去挖掘。

**以下为本次对话实录：**

**预言家游报：在本次展览中，贵公司是否有新IP产品发布？**

乐敏：这次很高兴能够出席中国授权展。阅文这次也是聚集了公司的头部 IP 来到展会。首先是位于展台正门的头部 IP 《大奉打更人》，一幅巨大的许七安的画面就能够展现出来《大奉打更人》出色的视觉形象，然后这个 IP 的影视作品我们预计在今年年底或是明年初会在腾讯视频上线。

除此之外，我们同时还带来了《全职高手》《狐妖小红娘》《一人之下》《庆余年》《斗破苍穹》《诡秘之主》和《道诡异仙》等。尤其是《道诡异仙》，一个新兴的以小说而闻名的IP，作品融合了中国民俗与道系美学，丰富了阅文IP的多样性。

**预言家游报：阅文旗下的 IP 产品今年在市场上的表现怎么样？**

乐敏：今年暑假，阅文 IP 大举挺进南京东路，从世茂的《全职高手》线下活动到 IP Box《一人之下》的线下曝光，以及到了百联ZX，我们跟塔斯汀进行了《狐妖小红娘》IP合作。在整条南京东路上面，我们都能看到阅文IP在线下有很好的表现。

不仅是在上海的南京东路，全国各地都有以《全职高手》《诡秘之主》《庆余年》《一人之下》等等头部 IP 做的线下落地的活动，给消费者和用户带来了很丰富的体验。

**预言家游报：如果从阅文自身出发，在未来的一年里，阅文打算去往哪一些 IP 衍生品的品类去扩展自己？**

乐敏：二次元的核心品类谷子，包括吧唧、色纸、亚克力牌等，是我们持续重点布局的品类。在这些主品类的授权方向上，我们也会增加更多的二创的图稿，让更多的用户体验到不同风格的 IP 形式的展现。

除此之外，我们同样也会在粉丝热衷的快消品，比方说连锁茶饮/餐饮、食品饮料这些领域去做大力的拓展。在3C、数码通讯类以及线下的实体娱乐类，包括剧本杀、舞台剧这一类的品类，也是我们比较看重落地布局的领域。这样我们能从各个维度触达二次元的核心人群，希望他们能体验到阅文IP的乐趣。

**预言家游报：您觉得过去一年IP授权领域有哪些新变化和趋势？**

乐敏：说到这个就说到了毛绒了， Jellycat 最近就非常火爆，带来了这么多的情绪价值给到用户，在当前社会环境中疏导出来情绪价值还是很重要的，所以我是觉得下一个风口可能是在毛绒这个品类。

同时毛绒这个品类其实对于二次元IP来说是很难去实现的。毛绒的脸部的开模是很难去用棉花把它塑形出来的。现在随着工艺上面的不断地提升，那么我们就看到了Labubu这种搪胶毛绒系列出现。一些新工艺的出现，使得阅文IP在毛绒材质的品类上面有了很大的突破。

值得一提的是，我们跟行业里比较头部的搪胶毛绒品牌超级元气工厂达成了合作，阅文IP授权的搪胶毛绒盲盒有望在今年年底或明年年初跟大家见面。所以我觉得下一个赛道可能是毛绒这个材质，一是能够安抚人心，让柔软的材质跟消费者有这样的一个互动，第二又能把喜欢的IP赋予在这个品类上面，让用户体验得到提升。

**预言家游报：最后，从阅文自身出发，请分享一下您对品牌授权市场未来趋势的看法或预测。**

我觉得二次元是个大主流，因为二次元人群越来越多，二次元的文化的属性和内容也越来越深度。你可以看到，很多商业体都在转型，很多街区也都在转型。国漫这个赛道也受到了非常多的品牌方和客户的认可。

所以在这一领域，我觉得24年才是一个真正的开始，我们真正将国漫内容去触达到全国各地的消费者，也是一个非常大的趋势。国漫是一个比较好，并且很有潜力能够传扬出去的IP内容载体。
