# 郭靖宇：被短视频切片偷走90%流量，原创IP不要用贵的明星丨实录

2026年5月13日，北京。在2026首都视听节目春交会上，长信传媒董事长、著名监制、编剧、导演、首都视听产业协会副会长郭靖宇发表了题为《剧向新程：从传奇叙事到原创IP，创作的变与不变》的主旨演讲。作为深耕行业多年、一手缔造无数爆款剧集的“写戏人”，郭靖宇以其标志性的直率风格，深入剖析了长视频创作者在AI浪潮与短视频冲击下的生存之道。

演讲开场，郭靖宇一如既往地接地气，自嘲“写演讲稿是不给钱的，写剧本是给钱的”，所以只准备了一页提纲。他首先坦诚地拆解了行业目前面临的三大“变化”。

![图片](http://mmbiz.qpic.cn/sz_mmbiz_jpg/7PwOwGuzgVAFuIdEeu7uQqiahbDO1EiayKYnPNp2pTprM1CrjotTZ3uV8OWAe5ZmVvd9fVAglPticxabmMjRsc1fLPEYzs1k2zO7GaegoVhVDg/640?wx_fmt=jpeg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

首先是播出载体的剧变。从舞台、银幕、电视台到长视频平台，再到竖屏甚至AI剧，郭靖宇感慨，“最大的变化就是很多人是用手机看的，这是创作者必须面对的现实”。

其次是市场环境的遇冷。他直言不讳地指出，现在支持行业的钱越来越少了，部分原因是行业自身的不努力，部分则是由于资本此前看好的产业如网络电影“突然一下子被整垮了”，导致资本信心受损。

最令他痛心的则是竞争关系的恶化。他认为，现在的对手不再是同行，而是游戏和短视频。尤其是短视频平台对长内容的“切片”式侵权，让他深感愤怒：“我觉得盗版就是盗版，终归要把版权的费用和正常的使用权还给我们……他们抢走我们90%的流量，能给我们百分之二三十也是可以的。”

尽管行业环境复杂，郭靖宇却展现出了老兵的定力，他认为创作的核心方向始终未变。他以自己的创作历程为例，从早期传奇剧推崇的“忠孝节义”，到《唐朝诡事录》系列的盛世古典与正邪诡异，内核始终是中华传统文脉。他甚至开玩笑说，以前自己被评价为只拍“忠孝节义、传宗接代”，但在他看来，这恰恰是民族文化中最简约的表达。

谈及新项目《烟雨神游记》，郭靖宇展示了如何完成“从传奇叙事到古典怀想”的升级。这个故事取材于唐诗、陶渊明的《桃花源记》及曹植的《洛神赋》等经典，试图用更符合年轻审美的叙事去触达观众。他坚信观众的审美正在变强，“写到非常小众、非常偏的历史文化思考，都有人能理解，都能找到知音”。

对于当下最火热的AI技术，郭靖宇展现出极高的拥抱热度。他回忆起24岁那年被老导演质疑“拍电视剧不是艺术”的往事，强调创作者不能固步自封。他认为AI是赋能工具，“AI应该是帮助聪明的、有艺术理想的人完成理想，增加我们的能力”。他甚至大方分享实操经验：写圣旨、查词汇都会问AI，虽然AI写诗词还“差点意思”，但在提高剧本生产效率上大有可为。

在原创IP的开发打法上，郭靖宇分享了他的“省钱经”与“系列化”逻辑。他明确表示：“我很少用那个非常贵的流量明星”，而是倾向于选择20岁的年轻人，并让他们在多季轮回中保持主创和演员的不变。他直言，这样做是为了对标奈飞，通过IP系列化开发来增强会员粘性，降低观众的陌生感。

演讲最后，郭靖宇回归创作者本色。他说：“创，就是个原创。别人说原创是不是太难了？其实也没有那么难……只要你脑洞大，不要闭门造车。”

面对行业的迷茫，这位曾因不满行业规则而主动退出的“乐天派”表达了极大的信心。他动情地呼吁同行：“我们行业又不是保家卫国那么难。我们不就是想护卫我们影视行业，想护卫我们自己的手艺，想护卫我们这赖以生存的这个行业，我相信一定能。”
