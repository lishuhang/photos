# 奈雪乐园，涨割股民，跌割会员

奈雪的茶再整新活“股票交易”系统，涨了皆大欢喜，你喝奶茶，我割韭菜，跌了降本增效。

奈雪在营销这事上一直跟得上潮流，元宇宙大火，他搞了虚拟人NAYUKI，股票连涨3天，一大批散户嗷嗷叫进去，什么股市之王、行业之星，茶饮之光，但随后从最高18元跌到现在的6元，这帮人又只好提着裤衩骂骂咧咧出来。

没想到连可动虚拟人还没有的奈雪，这一次玩的更妖娆了，6月30日上线了“股票交易”，用消费积分“奈雪币”为交易资金，与港交所实时绑定，只能购买奈雪的茶这一只股票。为了刺激大家购买奈雪币。除了兑换奶茶这样的常规操作，还增加了巨额奖励，比如苹果电脑，看似是一场资本让利、跪舔用户的送羊毛行为。

不不不，朋友们，事情没那么简单，资本是趋利的。

奈雪的这个虚拟股市只能买涨不能买跌，而且加了杠杆后还能被平仓，也就是说，假设你买了一万个奈雪币，变相的绑定了个人资产与奈雪的真实股价，不然按照他的杠杆倍数，分分钟爆仓。而让奈雪真实股价上涨的方法，无外乎买股票贡献流动性，或者买奶茶贡献业绩，这么想起来，是不是又成了一个完美的金融闭环？

搞金融化营销奈雪不是独创，他的前辈们要么吃灰、要么雷区试探、要么隐秘生长。
