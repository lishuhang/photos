今年春节，过节的不只是人，还有宠物。

对不少人来说，宠物是家庭的重要组成部分，一些养宠者甚至把宠物当成了自己的子女。“让宠物过个好年”，支撑起了春节期间宠物消费的火爆。

不仅主人过年要吃年夜饭、贴春联，猫猫狗狗也同样被安排了过年的“仪式感”。

在电商平台上，不少“宠物年夜饭”销量不菲。各种用面粉、肉类、水产制成的饺子、点心和菜肴，配上精美的礼盒包装，让许多网友惊呼“宠物吃得比人都好”。

宠物春联的销量更是“异军突起”。有数据显示，宠物春联节前销售额增幅为489%。

此外，由于春节宠物店休假、快递物流延迟等原因，在年前购买宠物食品和生活用品、带宠物“洗剪吹”、购买宠物新年衣服等，都是养宠者必须考虑的事情。同时，伴随春运返乡大潮，不少养宠者还需要考虑宠物寄养、代喂养等事宜。

**春节上门喂宠物，一天200元**

对于异地过年的宠物主来说，每年春节除了考虑抢火车票、置办年货，还需要妥善地安排宠物。因此，在每年春节，宠物寄养和上门喂养等服务会迎来一波需求高峰。

相比于鼠、蜥蜴、鱼等有较小空间就能生存，且日常不需要过多互动的宠物，猫、狗等宠物体型较大而又活跃，需要运动空间，也需要更多照顾与陪伴。

宠物寄养的价格并不统一，每天的费用从几十元到100多元不等，视寄养环境、宠物体型和是否有特殊需求而定。据了解，春节期间，宠物寄养费用会上调50%左右。

不过，受限于场地原因，能够提供优质服务的宠物店并不多，而更换环境容易导致宠物产生应激，或被传染疾病。因此，相当一部分宠物主会选择上门服务。

提供上门服务的，大多是住在本地，或就地过年，想在春节期间打零工的年轻人。他们有一定的饲养经验，负责定期上门喂食、换水、清理排泄物、遛狗等。假期前，他们会在QQ和微信群组、小红书、咸鱼等平台接单，与雇主协商价格。

上门服务大概费用按照当地消费水平、路上往返费用而定。一般来说，喂一次猫和清理排泄物仅需20-50元，而遛狗则需要根据犬只体型加价。如果在较大的城市，路程往返时间较长，宠物主还需报销车费。

居住北京，老家在东北的林虹每到过年就为寻找宠物养护服务焦头烂额。她告诉价值星球，因为许多往年在北京就地过年、提供上门喂养服务的年轻人回了家，今年很难找到合适的上门服务者，服务价格也更高了。

去年，林虹花了100元寻找寄养服务，今年因为提供服务的人变少，她家附近找不到合适人选，只得选择了一位较远的服务者，并需要报销路费。经过协商，她最终把价格定在了每天200元。

“我预计在家待10天左右，对方每两天去照顾一次，这样大概需要至少五次，合计一千块钱。”林虹说。

大城市的流动人口为宠物寄养、上门喂养等服务焦虑；住在中小城市，不需要回家过年的宠物主，则已提前为宠物预约好“洗剪吹”等新年美容服务。

在平时，一线城市宠物洗护价格可以达到100元以上，二、三线城市多为几十元，但在节前和春节假期内则会上涨约50%。即便如此，宠物仍然一“澡”难求，很多店家都需要提前预约。

一位从事宠物护理的工作人员告诉价值星球，平时休息日店内每天大约接待5至7只宠物做洗护，春节期间则要每天服务10只以上。“大家会扎堆带宠物做洗护，我们店里部分员工也要休假回去，留守员工每天连轴转，涨价才能给员工付加班费。”

**宠物过春节，春联年夜饭一样不少**

近两年，除了宠物寄养、上门照顾、洗护等刚需，满足宠物主精神需求的宠物对联、宠物年夜饭等服务也开始流行，并越来越火爆。

今年，养了两只猫的珊珊第二次为猫咪购置了春联。她计划把春联贴在毛绒绒的方形猫窝上——尽管猫咪平时很少钻进这个窝睡觉，但毕竟贴了春联后更有节日气氛。

“就是增加一些节日氛围，和家里其他的布置更协调嘛。猫虽然看不懂，但人看着开心啊！”珊珊说。

珊珊注意到，宠物对联和其他搞笑春联几乎在同一时间兴起。她认为，支撑起这一消费趋势的，正是大量在异地工作、独居，更追求新式审美的年轻人。

宠物春联或搞笑春联与传统春联一样，用红纸书写，内容却颇有反差感。它们大多使用可爱的艺术字而非传统书法，用诙谐的语言祝福新年快乐、工作顺利；有些甚至没有具体内容，只是“哈哈哈哈哈哈哈”或动物、心形、云朵等简笔画图案。

珊珊记得，她去年第一次买搞笑春联，是送给一位因担心感染新冠，选择就地过年而倍感沮丧的程序员朋友。这幅春联上联是“years months weeks day day no bug”，下联是“python java C++ line line so easy”，横批是“0 error 0 warning”，寓意程序员编程顺利、没有错误。朋友收到后哈哈大笑，心情愉快了许多。

在为朋友挑选搞笑春联的过程中，珊珊也留意到宠物春联，顺手给自家宠物买了一幅。

据价值星球观察，宠物对联的价格一般在5至15元区间，文案既有“愿汪成真”“喵宅纳福”等与宠物有关的吉庆内容，也有“餐餐有罐头、天天睡懒觉”等调侃内容，还有些则是把春联、福字内容全都改成了“汪”或“喵”字。这些对联的长度适合宠物笼、窝的大小，一些对联还饰有卡通宠物简笔画图案。

珊珊和父母住在一起，由于常有亲友来访，父母不希望她把无厘头的搞笑春联贴在门口，但支持她给猫窝贴对联，增加家里的喜庆气氛。而她也庆幸今年早早给猫买了一幅印着猫爪的对联——由于宠物春联的火爆，她在2月2日再次查询自己下单的店铺时，发现猫狗对联已经全部售罄，只剩一款横批“鼠圆玉润”的鼠类宠物对联有货。

销售火爆的除了宠物春联，还有一些商家推出的“宠物年夜饭”。这些用面粉、肉类、水产品做成的饺子、汤圆、点心和各类菜肴，构成了宠物主为自家宠物准备的一桌“过年大餐”。

在淘宝上，豪华配置的宠物年夜饭售价可高达数百元。在一家店铺销售的产品中，一百多元的宠物年夜饭套餐包括造型精美的龙形点心、龙角骨头、锦鲤奶冻等。虽然价格不菲，但店铺仍然生意火爆，多款产品显示售罄，商家在产品分类中加入了提示：“部分特供产品售罄无补/预计1月25日至31日发货。”

而在另一家店铺，标价400元的天价宠物年夜饭套餐中，菜肴甚至被做成了十道菜，汇聚“鲍参翅肚”等昂贵用料的模样，让许多网友惊呼“宠物吃得比人都好”。

但并非所有宠物年夜饭都如此奢华，电商平台上商家众多，为宠物购买年夜饭丰俭由人，30至40元的一顿饺子、60元左右的宠物佛跳墙、100元左右的精装礼盒比比皆是。

珊珊并不准备购买宠物年夜饭。她认为，这些食品不仅不经济实惠，对宠物健康也不一定有好处。例如，花花绿绿的点心大多加了色素；看起来添加较少的一份宠物佛跳墙要58元，她自己团购人吃的佛跳墙，每份也只需40元以内。“还不如给猫多开一盒金枪鱼罐头。”珊珊说。

不过，充满创意的宠物年夜饭仍然十分火爆。淘宝搜索显示，宠物年夜饭本周搜索量上升13%，热度达到573.6万。

**营销花样背后，宠物市场亟待规范**

宠物消费热背后，是近年中国宠物市场规模的稳步增长。

随着饲养宠物的年轻人越来越多，2023年，中国的宠物数量已经增长至2亿只，其中以猫、狗为主。

另外，由于大城市空间的限制、年轻人工作繁忙，缺少定期遛狗时间，宠物猫数量从2018年起高速增长，数量和增幅均超过宠物狗。

宠物主的年龄年轻化，也影响着宠物行业的消费趋势。派读宠业参考发布的《中国宠物行业白皮书——2022年中国宠物消费报告》显示，2022年，95后宠物饲养者达到36.8%，90后在宠物饲养者中占比超过50%。同时，高消费宠物饲养者主要集中在一线、新一线及省会城市。

大城市的年轻宠物饲养者消费能力较强，对宠物产品质量的要求也较高，往往倾向于选择较为营养健康的宠物主粮和零食，并会购入智能宠物用品。与此同时，年轻消费者也更倾向选择那些美观而更有创意的宠物玩具、装饰品、零食等。

春节，则成为宠物主消费需求集中释放的一段时间。但整体来看，伴随宠物产品与服务的多样化、个性化发展，整体宠物市场仍有待进一步规范化。

其中，宠物食品安全是消费者普遍较为关注的问题。长期以来，宠物食品标准不明确，仅有国家推荐标准而没有强制性标准，因此偷工减料、虚标成分甚至掺假的情况并不少见，容易导致宠物疾病。

截至2024年2月，黑猫投诉平台涉及猫粮的投诉有10558条，问题包括冻干潮湿发霉、虚假保质期、猫粮导致宠物猫呕吐或肝功能异常等；涉及狗粮的投诉3223条，问题包括狗粮发霉、毒狗粮导致宠物狗腹泻甚至死亡等。

除了宠物食品，一些加入所谓“黑科技”的宠物智能用品，如自动猫砂盆、智能饮水机、智能加热猫窝等，同样在养宠圈内引发了较大争议——近年来，时有智能饮水机漏电、智能加热猫窝倾倒闷死猫、自动猫砂盆卡死猫等悲剧发生。

此外，一些宠物店同样因缺乏监管，导致宠物在过度繁育和拥挤饲养环境下性格异常，甚至携带猫传染性腹膜炎等致命疾病。在寄养过程中，这种环境也会影响宠物性格、危及宠物健康。由于寄养、上门喂养等服务缺乏监督，宠物主往往要花费大量时间精力与服务提供者“斗智斗勇”。

家在南京的许女士曾经有过很不愉快的寄养体验。她在全家出门旅游时，把饲养的哈士奇寄养在宠物店。尽管她对宠物店的环境满意，但工作人员并未完全按约定时间遛狗，导致她的狗精力无处发泄，“回家后到处搞破坏，出门就狂跑，一看就知道遛得不够”。此后，许女士每次出门，都自驾带狗出行，或托信任的朋友照顾。

上门代喂养服务则更加缺乏保障，由于双方只通过网络平台达成交易，而服务涉及到家中财产安全和宠物健康，饲养者不得不想尽办法保障自身合法权益。

林虹每次联络上门宠物照看服务前，都会想尽办法确认对方的身份——她只找养过猫的人，要求对方发和猫相处的照片，提供手持身份证、电话、工作单位等个人信息的录像，并花几百元在家里装上了全方位无死角的摄像头。“只能先小人后君子了，不然万一猫出了事情，或者家里丢东西，花的钱和精力就更多了。”她说。

为了解决这些宠物行业内部的乱象，业内人士正在不断做出努力。

2023年，有电商平台推出“宠物食品原料透明化”项目，通过审核评选出诚信品牌。在托管照顾方面，有互联网行业宠物爱好者成立了上门喂养中介平台，服务涉及一、二线城市和部分三线城市，通过面试招聘、收取保证金等方式，雇佣宠物托管照顾人员。同时，专家学者也开始为宠物食品安全立法提出呼吁。

随着宠物行业市场规模的增长，商品、服务逐渐规范化是不可阻挡的趋势。只有当行业生态更加健康，让消费者发自内心放心、满意，中国的宠物商品才有机会获得更大的市场份额，乃至走出国门，进入更广阔的国际市场。
