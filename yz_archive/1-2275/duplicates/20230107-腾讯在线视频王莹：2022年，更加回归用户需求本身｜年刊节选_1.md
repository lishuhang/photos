**刊前语**

2022对每一个中国人而言，都是最特殊的一年，从千磨万击，到砥砺前行。尤其是文娱行业：长短相接、降本增效，跨媒融合，现在文艺创作和时代精神的关系更为密切，在这个环境下仍有不少人坚持“咬定青山不放松”。为此，娱乐资本论选择在2023年度CEIS娱乐产业年会之际推出一本专刊，与14位业内大咖展开对话，探寻新时代的生存法则，致敬行业坚守的价值初心。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXxicEMLvhzW2AqTCZZtx7ybZj0ttKyD9aeUXCJUm2O2t3gLxEoicRjYgUaCHbK0rcC9ZB9jqiaYBxEpCg/640?wx_fmt=png)

当降本增效的风潮席卷行业，与项目总数大幅度下滑对应的是头部内容价值的进一步飙升。这不仅意味着优质IP的争夺愈发激烈，也意味着IP的长效价值挖掘、复合开发成为长视频平台下一阶段的关键课题。

借助不同内容形态开发实现用户扩容、借助系列化开发提高用户粘性、借助IP授权实现多元商业变现。围绕IP能讲述的故事正变得越来越多。

腾讯视频无疑是其中的佼佼者。就拿其暑期热播大剧《梦华录》为例，不仅创造了超过40家品牌的合作记录，更是依靠和喜茶、奈雪的茶、十三余等品牌的联动大热出圈，直接火到了新消费领域。

同时，今年，腾讯视频IP《斗罗大陆》、《吞噬星空》、《完美世界》也在动画这一内容形态上表现亮眼，为IP的多元商业化开发提供了更多维度的内容视角。

![](https://mmbiz.qpic.cn/mmbiz_jpg/jNZszpkibXxicEMLvhzW2AqTCZZtx7ybZjMQFVRwp86BpFQnKZQjOL8kfIfYKkPo9oyLjYNkHvgxUDM4frxLzfjg/640?wx_fmt=jpeg)

作为长视频领域的“IP”大户，腾讯视频当前对IP的增值开发处于什么阶段，有怎样的经验可以分享？对于以《斗罗大陆》为代表的IP长线开发，又有着怎样的思考？带着这些问题，娱乐资本论对话了腾讯在线视频平台增长和IP增值部总经理、《斗罗大陆》IP总负责人王莹，尝试找到答案。

![](https://mmbiz.qpic.cn/mmbiz_jpg/jNZszpkibXxicEMLvhzW2AqTCZZtx7ybZjpIBT0NB3TPGyv0NEOZAlpB5zWz2y4ich0DBy3cznFEEdN6FEmtgoJ6A/640?wx_fmt=jpeg)

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7tMuAmiadsyvAk4l30TZvmgP03RGX0iaosuL5yVawsdblYqeWUcOTHYoQ/640?wx_fmt=png)**回归用户需求，用复合开发为IP做加法**

2022年的种种挑战被王莹视为摒弃泡沫、回归价值初心的契机，这也正是她对于“咬定青山”四字的理解。从长期主义角度出发，这样的回归并不算是一种坏事。

对腾讯视频而言，永远锚定用户需求的坚持始终清晰。具体到IP开发领域，便是不断挖掘用户不同阶段对IP的需求，借助多元化的内容开发，将IP价值带上新的台阶。

娱乐资本论：您如何理解这次年刊的主题——“咬定青山”这四个字的含义？

王莹：所谓咬定青山，就是无论外部的环境是顺风、逆风，不管有多大的挑战，关键看清楚我们要坚持什么。我觉得对于腾讯视频来说这是非常清晰的。任何时候我们永远锚定用户需求。我们要持续挖掘和洞察用户新的需求，去创作他们喜爱的精品内容和内容的衍生形态。这在什么时候都是最重要的，也是所有事情的出发点。

虽然今年的市场非常有挑战性，但也能让我们在一定程度上抛开一些虚的东西，更加回归到做事情的价值本身。

娱乐资本论：在一定程度上可以说更务实了？

王莹：对。包括最近聊的比较多的，用户对于IP 的需求到底是什么？我们当下把一个用户对 IP 的需求拆分成几个层次：最基础是他有“看”的需求，然后用户看完后就有了“聊”的需求。我们现在做的IP增值业务，不管是衍生品还是线下，其实满足的是用户对于IP “买”和“玩”的需求。

所谓务实，其实就是回到用户需求本身，不断深入挖掘他不同层次、不同阶段的需求，再根据他的需求去做他觉得有价值，也能够更好放大 IP 影响力和商业价值的事情。

娱乐资本论：今年有很多IP跨界出圈的案例，您是如何理解ip的潜在价值和增量空间的？

王莹：首先我觉得 IP 的待挖掘空间很大，所有的事情都才刚刚开始。我们对 IP潜在价值的理解，最早是用户要来付费订阅。当他在这里投注注意力和时长，我们去做广告变现。现在当用户希望不仅能在屏幕后观看这些热爱的内容，还要让IP打破次元壁来到真实生活中。此外，他还想去变成 IP 的一部分。

对于我们来说，更重要的是如何用不同的产品形式来承接用户的需求，又做出一些超越传统预期的玩法，这样自然也就形成了出圈的案例。

总体来说，我觉得这是一个不可逆的、用户需求发展的趋势。

娱乐资本论：目前腾讯视频内包括斗罗大陆、斗破苍穹、 狐妖小红娘、鬼吹灯系列等都在进行多种内容形态的复合开发，您认为IP在内容端的增值对后续商业化开发有什么价值？

王莹：一个IP内容持续的复合开发，有两个本质的动力。一是随着时间的推移，用户的兴趣和审美发生变化。所以像仙剑、金庸这些经典IP会持续去做新版本。另一个是技术发展可以带来创新形态内容开发的可能性。比如现在看的是平面内容，未来也许会做 VR 内容，或者做数字人、元宇宙。

而不同的内容品类，哪怕只是当下主流的内容品类，比如动画和真人剧集，人群就非常不同。动画整体人群还是偏男性、更年轻。剧集受众就更广谱，女性观众特别多。比如《斗罗大陆》，我们做了动画，5年不断更，已经积累了庞大的粉丝群体。但去年真人剧集上线的时候，就看到这两个品类的人群重合度实际上小于一半。《斗罗大陆》动画五年来积累了两三亿的动画粉丝，但剧集一上来就是一亿以上的用户，这部分用户大多是原来不看动画的。

所以实际上不同的内容形态开发是对不同的用户人群做渗透。不同媒介形式对IP叙事乃至各方面的能力加持是不一样的。比如动画，特别燃、炸，很多真人拍不出来的东西动画是可以实现的，它充分满足了人们的想象力。但是剧集可以做更深刻的人物刻画，它在叙事上更饱满，在这方面动画很难替代。所以我们在做长线 IP 开发时，会充分考虑到这种内容形态的特点和对应的人群需求，以及在叙事方面的一些独特性，把 IP 的用户渗透和影响力做到最大，从而使IP的商业价值也保持持续增值。

还有一点，不同品类在持续开发过程中有一个配合，所以整个 IP 的生命周期是可以延展的。

娱乐资本论：这个配合指的是可以在上线时间上打一些配合？可以有效的填充，保证 IP 以各种形态完成用户输出？

王莹：对，比如《斗罗大陆》，去年在动画年更的过程中，剧集上线时，用户的整个渗透率、喜爱度以及变现力有一个快速的拉升。如果说动画持续不断更是持续积累影响力的过程。当你叠加一种新的影响力巨大的内容形态时，IP整体的品牌价值是一个急速拉升的过程，直接把IP拉上下一个台阶。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7ejDSZf97dAE3mMYqSpwDp0blV0YsOONibSOjLz8EycRV8uxj7xc8QIg/640?wx_fmt=png)**亿级用户、商业价值爆炸，动画产品极大提升IP价值**

尽管提及IP开发，大多数用户的第一反应还是剧集，但动画这一充满想象力与生命力的内容形态也在近年来发挥着愈发关键且不可替代的作用。以腾讯视频的动画《斗罗大陆》为例，伴随五年不断更的内容开发，这一作品累计了亿级年轻粉丝，是一片具备旺盛消费力和高粘性的用户群。

据王莹透露，目前《斗罗大陆》动画的商业合作情况与腾讯视频站内的S级大剧处于同一水位。在她看来，这一IP具备和国际顶级IP同样的开发潜力。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXxicEMLvhzW2AqTCZZtx7ybZjuXboYrGyBft1EicaXKUIVUECtuzA5hrqbrFxaXEpAfwqvgl1bVXLMUg/640?wx_fmt=png)

娱乐资本论：以斗罗大陆IP为例，数年来这部番剧已形成了稳定的年更模式，您如何看待这一IP的商业价值和用户影响力？

王莹：《斗罗大陆》的小说创作于2008年，腾讯视频 2018 年上线了动画，持续做年番，不断更，今年已经是第五年了。五年下来，动画用户覆盖破亿，整体播放量也超过 400 亿。在整个国内的动画领域里是绝对的领先状态。

众所周知，动画对于《斗罗大陆》IP 的起飞起到了至关重要的作用。本身斗罗IP的故事非常热血励志、世界观极具想象力，设定有趣，团魂很燃，但如果只停留在文字层面，很多东西大家只能自己脑补。而动画最大的价值就是把一个世界实实在在呈现到了用户面前。

同时5 年的不断更也是一个持续累积粉丝和IP影响力的过程。从这个角度来说，不管是在平台上的商业变现，还是《斗罗大陆》的 IP 授权，这几年都是一个快速爆发式增长的过程。玩具、食品、饮料、服饰周边、3C、游戏、出版等生活场景里的各个品类，其实都有《斗罗》的衍生品出现。

娱乐资本论：那《斗罗大陆》动画在广告领域的商业合作大概能达到哪种剧集的定位？

王莹：完全是s 级的。

娱乐资本论：那您觉得《斗罗大陆》 IP 背后还有哪些想要去挖掘的增量？

王莹：我必须得说，不是所有的 IP 都可以常年持续挖掘的。但是对于斗罗这样的 IP 来说，确实有很大的潜在增量空间。因为第一，整个IP本身故事的容载量很大，仅是用不同品类把这个故事讲一遍就需要很久。目前我们动画做了5年，连小说第一部都没更完。而这个 IP 一共有五部小说，四部外传，本身它的故事就有很大的开发体量。

第二，它的世界观是有极大的辨识度和差异度的，在其中做衍生创作的空间也是巨大的。我们可以从它的世界观里延展出新的人物和故事，延展出更多基于现在用户的审美和兴趣、基于社会热点的内容。同时，媒介技术发展速度飞快，除了讲故事，还可以布局数字人、VR、线下等新媒体艺术项目。

娱乐资本论：您预估这一 IP 的潜在商业价值具有怎样的规模？

王莹：我们自己的理想还是对标世界上最头部的IP宇宙。至少《斗罗大陆》是有这样的潜力的，重要的是要把它做出来。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7gMbSIRF8ujdpJibC3CLgiaEEY6kJq4YuKUC4cv1ZG4kjEVEHhs35Zn3Q/640?wx_fmt=png)**体系化参与商业合作过程，IP授权不做消耗做增值**

2022年，腾讯视频全IP的衍生品授权sku已经超过了2500个，品类涉及各个生活场景。商业授权无疑是IP C端价值中的重要拼图。

在王莹看来，合适的IP授权不仅不会消耗用户的喜爱，反而能推动IP本身的发展，但这也对平台提出了更高的需求：需要找到高度契合的合作伙伴，足够过硬的产品是底线，同时，平台方作为IP持有者，需要参与授权产品设计、营销的各个环节，进行深度品质把控。

娱乐资本论：从IP增值角度理解，您认为怎样的商业授权能够为IP实现赋能而不是消耗？

王莹：有一个很重要的标尺：授权是否在发展 IP 的内核，在持续累积用户对 IP 的热爱，还是在消耗用户对于 IP 的热爱。我们有一套方法论做保障。

IP授权是一个很体系化的工作，整个周期比较长，它涉及到实体商品的生产、销售和设计。所以第一，能不能选到合适的合作伙伴很重要。

所谓合适，是我们期待合作伙伴跟我们对 IP 的价值观、内核认知有强共识。不管这个商品是虚拟的还是实体的，它更重要的在于实现用户共鸣，满足消费者需求，加强用户的互动，能把用户对IP的爱延续，打破屏幕的限制把IP带到用户生活中。

第二，在这种共识的前提下，腾讯视频通过系列化的参与动作来确保授权合作质量。首先，我们会深度参与到整个IP衍生品的设计链条中，帮助被授权的合作伙伴。举个例子，我们现在的IP设计监修团队中有很多方向的设计师，比如服装设计师、潮玩设计师等，各个品类都能和被授权商直接对话，这些专业的设计师会给到针对性的开发思路和图库使用说明以及监修指导，确保授权产品跟 IP 的调性一致。

然后，在商品开发出来之后，在营销和体验层面，我们也会给到合作伙伴很多推广建议。作为IP方同时也是播出平台，我们在 IP 播出过程中，积累了一手的用户信息反馈和实时分析。我们会把这些信息也及时同步给合作伙伴，并做一些平台层面的积极联动，帮助他们更好的推广和营销产品及品牌理念。

最后，腾讯视频线上平台有草场地官方衍生品商城，线下未来也会有越来越多的IP实景体验触点，这都是非常好的商品销售场景。在这些场景中，我们也会跟合作伙伴联动，共同触达 IP 的消费者，促进商品的销售。

所以其实不管是《斗罗大陆》或是《梦华录》，都是用这样的全链路思路去开发的。未来，我们还会在更多 IP 上套用现在相对比较成熟的授权方法论，来实现对 IP 的赋能和对被授权合作伙伴的加分。

娱乐资本论：我们对于这种授权商品本身的影响力或品质，会不会有一个预先的把控？

王莹：有的，而且是一个强把控，这是底线要求。授权商品它本身是一个好的产品是非常重要的，所以在做合作伙伴选取时，对于潜在合作伙伴以往产品的品质，包括制造端、供应链端，我们都会进行考察，确保这首先是一个过硬的企业制造商。我们上次做《梦华录》衍生品的时候，就有很多用户反馈说他们还蛮惊喜的，认为衍生品质量超出预期。

娱乐资本论：那衍生品开发领域，除了目前的文创周边、手办外，腾讯视频未来有哪些新的计划？

王莹：我们现在全IP衍生品的SKU达到了 2500+个。从可以追踪到的角度看，今年购买我们衍生品的用户已经超过 1000 万，所以接下来还是要继续把它做深做实。一方面平台有很多长线 IP ，像《斗罗大陆》、《完美世界》、《吞噬星空》，还有包括金庸系列、仙剑系列、鬼吹灯系列等，另外我们每年也有很多新的热门IP 上线，除了剧和动漫，还有像《风味人间》这样的纪录片，都有很大的开发空间。所以其实对我们来说，更多是覆盖有潜力的IP，把现在做的品类进一步渗透。当然品类上也做了聚焦，会做对 IP 有强加持的衍生品。

第二就是在整合营销推广上，会更深入和品牌合作伙伴进行联动。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx9WDk0VsktjFDw7uoF1hD9wfcEvibWUNhlN8DDFb0XDHo3EKQL2hVB3tP8Xibib0Nib83yUoXgct3vQHg/640?wx_fmt=png)**IP落地线下大势所趋，跑通模式比追求收入更重要**

随着线下实景娱乐的发展，IP结合文旅产业，赋能线下业态也成为热议话题。今年《梦华录》播出后，长沙大悦城与这一IP合作的古风集市便引发大量游客前来打卡，也获得了行业的高度关注。

![](https://mmbiz.qpic.cn/mmbiz_jpg/jNZszpkibXxicEMLvhzW2AqTCZZtx7ybZjPTjxjS5twrnpqBbTqg9A2hzSR8tyfTvveUPicyP2KicX8VzYELRXiaF9w/640?wx_fmt=jpeg)

据王莹介绍，未来，IP与线下业态的联动将是腾讯视频持续探索的课题。在她看来，这样的合作具备强双向加持作用和巨大的挖掘空间。今年和明年，腾讯视频会尝试跑通一条可规模化、可复制的商业模式。

娱乐资本论：今年梦华录和长沙的线下联动也是IP和文旅结合的案例，您如何看待IP和线下业态的双向赋能？

王莹：我是特别看好的。本质上用户需求是在的，他们想成为IP故事的一员。而现在，我们可以在城市里把IP变成用户生活的一部分。最近看到行业专业评论说，未来所有的商业都会做的更加体验式。那在这个过程中 ，IP 是一个极好的连接器。

另一方面，原来 IP 更多停留在隔着屏幕“看”的阶段，所以现在线下的业态实际上是能够满足用户原本被压抑的“玩”和沉浸式体验的需求。从这个角度来说，实景娱乐体验对于 IP 的影响力和用户喜爱度的提升都有很强的加持。

更重要的是，线下衍生出了更多的商业潜力。每一个线下的项目都是一个销售的场域，可以卖商品、卖体验、做品牌合作。而且它不仅是一个商业产品，比如我们《梦华录》这个项目，一个很大的意义就是引发了大家对于宋朝传统文化的热爱和好奇。所以 IP 的线下业态也有很强的社会价值。

所以总体来说，从用户的需求释放到对于线下实体产业升级的推动赋能，从用户价值到商业价值再到社会价值，IP线下产业都有很多可挖掘的空间。

娱乐资本论：您理解什么样的 IP 比较适合和线下业态做结合？

王莹：第一，它要有较强的辨识度，跟现实生活有较强的差异化。比如二次元文化、古风文化类的 IP，就会更容易引发用户沉浸式体验的需求。这个辨识度不光是视觉上的，还有设定上的。比如《鬼吹灯》，就是任务冒险式的，可以设计线下的冒险体验，它本身的设定就非常值得互动。

第二，它也要有一定的用户影响力，否则如果需要用户从 0 开始认识，就比较难起到吸睛和引流效果。

娱乐资本论：IP和线下结合的最大的痛点在于切实完成用户喜爱的沉浸式落地，当前腾讯视频是否寻觅到了解决方案？

王莹：我们打造沉浸式体验项目会拆分成几个维度。第一是感官的体验，第二是功能的维度，第三是社交的维度，第四个是自然的维度。自然指的是因地制宜，今年暑期，我们的儿童IP小企鹅乐园和广东的一个露营品牌进行线下结合，就把露营地的风景和小企鹅的故事体验做了结合，这就是自然维度。还有一个文化维度。我们会从这几个不同的角度来整体评价消费者在一场体验里的认知和感受，在进行设计的时候，会去考量是不是在每个维度做到了极致。

另外，一支专业能力过硬，能够进行全案IP线下实景娱乐体验项目交付和运营的团队也是非常重要的，腾讯视频今年也打造了一支这样的团队。《梦华录》长沙古风市集就是这个团队的第一个作品。

娱乐资本论：目前在线下业态的开发上，腾讯视频更看重影响力还是商业回馈？

王莹：没有先后顺序，但我们现在第一步不是追求收入规模。对我们来说做好产品还是最重要的。比如我们未来可能会做《梦华录》的宋朝文化市集巡展。做这一场活动，就要把它做得扎扎实实，从用户的角度和商业模式上都跑通，且它未来是可复制可规模化的。对我们而言，过去一年和明年都是打磨产品，打磨商业模式的阶段，重要性高过做收入规模。有点像回到第一个问题，咬定青山。对我们来说，还是先把价值做实。
