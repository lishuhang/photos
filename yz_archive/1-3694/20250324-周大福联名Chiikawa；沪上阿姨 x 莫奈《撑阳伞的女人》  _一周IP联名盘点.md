**库迪咖啡 x 哪吒之魔童闹海**

## 图片

2月27日，库迪咖啡官宣与《哪吒之魔童闹海》达成“现制咖啡饮品”领域独家合作。3月17日正式上线联名活动。此次联名推出魔丸、灵珠系列共7款饮品，且有丰富周边，包括限量版哪吒徽章、冰箱贴、吧唧、卡片等，还有极具特色的联名主题包装，热饮和冷饮杯均采用哪吒主题设计，消费者购买任一饮品即可免费获得联名杯套、杯子和纸袋。

**新世纪福音战士 x 潮玩星球**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

2025年3月，潮玩星球与《新世纪福音战士》展开重磅联名。3月21日至5月31日，上海、广州、武汉同期开放主题餐厅。餐厅供应8款主食、8款甜品、10款饮品，消费任意饮品或甜品每满76元，可得随机PET杯垫一张；消费任意主食，可获随机PVC餐垫一张。同时，主题店周边于3月21日上架贩售，包括徽章盒蛋、立绘立牌、透卡等多种周边。

**周大福 x Chiikawa**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

3月23日，周大福与Chiikawa推出联名黄金产品，包括6款黄金串饰及3款金章。香港尖沙咀iSQUARE国际广场店、澳门美高梅店、深圳壹方城购物中心店、成都万象城店及上海550新形象店率先发售，每人限购3件，售完即止。发售当日早上8时，店外就排起长队，10时半正式发售，不足2小时便有4款商品售罄。

**世界之外 x 罗森**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

3月24日至4月20日，《世界之外》与罗森展开联动活动。3月24日起，消费者前往罗森门店，满足集点规则，即可兑换1份【世界之外×罗森联动行李牌】。活动期间，线上渠道于3月24日10：00开启，用户可搜索罗森ip购小程序或罗森京东自营旗舰店选购巧克力燕麦脆、茶包等食玩产品，以及马口铁徽章、色纸、摇摇乐和毛绒等联动周边。

**无限王者团 x 羊城通**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

3月14日，羊城通与源自《王者荣耀》的虚拟偶像男子团体无限王者团达成联名合作。其一，【无限王者团Call Your Name】系列实体珍藏交通卡正式开启预售。其二，手机交通卡方面，【无限王者团 无限奇迹之夜】系列虚拟卡面皮肤同步上线岭南通APP的卡版商城。

**必胜客 x 蔚蓝档案**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

3月20日至4月13日，必胜客与《蔚蓝档案》开启联动活动。通过指定渠道购买品牌主题联动套餐，即可获相应联动实物周边与虚拟道具礼包。购买蔚蓝档案多人餐，可得「多插亚克力立牌」「吧唧套组」「炫彩镭射票套组」各1份及「必胜奇迹礼包」虚拟道具礼包1份 。

**沪上阿姨 x 莫奈《撑阳伞的女人》**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

3月19日，沪上阿姨开启与莫奈《撑阳伞的女人》的联名活动。此次联名推出青苹果之春、春日龙井青团、春光浮现三款饮品低。活动期间，还上线了印象春日纸杯、纸袋、杯套等充满艺术感的包材，以及春日透镜、春日油画猫窝等特色周边。此外，3月19日至25日限量推出含猫窝和两杯春日限定饮品的「春日油画猫窝周边套餐」 。

**泡泡玛特 x 天线宝宝**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

3月20日，泡泡玛特与天线宝宝推出联名产品，并于线上平台率先发售，次日（3月21日）开启线下发售。此次联名盲盒采用贴毛工艺，造型适合制作挂件。此外，3月22日在上海五角场合生汇1F举办天线宝宝见面会，会员在门店购买任意天线宝宝系列产品，可获与天线宝宝偶装互动机会。

**Keep × 初音未来2.0**

初音未来 x KEEP 主题线上跑【第二弹】火花之约现已上线，其活动时间为2025年3月21日-2025年5月20日。此次活动推出了 “花火之约” 经典款和趣玩款奖牌。趣玩款奖牌能投影 MIKU 剪影，定格舞台瞬间，自带支架可摆放收藏，采用滴胶 + 闪粉工艺，还原盛典上流光溢彩的夜空。

**瑞幸咖啡 x wiggle wiggle**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

3月17日，瑞幸咖啡与韩国文创品牌Wiggle Wiggle展开春日季联名活动。此次活动期间不仅上新了3款联名纸袋、2款联名杯套，还推出如「一起春游BlingBling贴纸」「春日瑞一杯徽章」「踏春拎拎保温杯」等周边，消费者购买指定套餐即可获赠。活动期间，珞珈樱花拿铁升级回归，瑞幸线下还焕新了联名限定主题店，邀请消费者打卡。
