**作者|吴立湘**

大家好，我是娱乐资本论吴立湘。欢迎大家来到由徐汇区文化和旅游局指导，界面新闻、娱乐资本论、徐汇文化旅游服务和发展中心共同主办的“首届上海文娱科创沙龙”。

今天我的演讲主题是《文娱行业的多模态战争》，这是我们成立AI生态小组这三个月以来，通过拜访30多家AI公司、20多家文娱公司所积累的一个观察。

不过，我首先将向大家介绍一份备受期待的榜单——《中国“AIGC×泛内容”行业概况+50家最有价值公司》。

这份榜单是根据我们对中国人工智能与泛内容领域的调研和分析得出的。我们将这个庞大而新月异的行业划分为五大类：

模型类、模态类、文娱公司AI转型类、虚拟人/虚拟驱动类以及AI场景赋能（AIaaS）类。

![](https://mmbiz.qpic.cn/mmbiz_png/UgtzVuzhFd5COeWGy9o07bERnkPCj6kA7CxyXVMNvT366piaetM8wlPjNpzqDWcb6QyMju5aImKBSzLSz03EMOQ/640?wx_fmt=png)

在这些50个名单中，我们探寻了其中30多家公司，并进行了综合性报道，同时横评了5家大模型公司，接下来随着其他40多家大模型的发布，我们也将更新自己的评测列表。

我们欣喜地看到，在大模型领域，中国虽然与海外仍存在一定差距，但追赶的步伐正在加速；中国的数字人公司，随着新一代NLP的发展，涌现出了更好的交互、更全面的智能和更有效的场景；更令人振奋的是，中国在应用层和模型层涌现出大量具有人工智能背景的创业公司，他们能够更有针对性地服务于中国企业和产业。

我们还发现，中国文娱公司在这一浪潮中的转型力度之大令人瞩目。有些公司成立了AI研究院，有些通过总体工作流优化提升了20-30%的效率，有些用AI给了新人创作者更多的机会。这些令人鼓舞的举措展示了中国内容企业和科技行业在人工智能领域的创新能力和融合决心，就像郭帆导演在我们沙龙所说的，中国影视业即将迈入工业化3.0时代。

![](https://mmbiz.qpic.cn/mmbiz_jpg/UgtzVuzhFd4Y6kOUvCeA4N85sib5p4QC7F1mUplz105ib9qcsuIqsylicWGYG1OORjbtribe8amv2rONr65LPL0sdg/640?wx_fmt=jpeg)

当然，我们也认识到之前媒体宣传过于夸大，实际上，AIGC是一个漫长的迭代过程。将其嵌入工作流中仍需要进行组织结构上的大规模优化。因此，我们决定每半年为节奏持续发布《中国AIGC50榜单》，以实时跟踪最有前景的公司，为大家提供最新的行业动态和发展趋势。

**主题演讲：**文娱行业，进入多模态战争****

# 在两个月前，如果我在现场列出【文娱行业，进入多模态战争】这个标题，我相信绝大多数人都会一脸懵，啥叫多模态？

# 

![](https://mmbiz.qpic.cn/mmbiz_jpg/UgtzVuzhFd5COeWGy9o07bERnkPCj6kAfzm9OmbZ8B01EdtZCOrCJ9HPH2j2JuIXIKQ47dXk9dUkVxGd2pkiaicg/640?wx_fmt=jpeg)

# 但随着AI浪潮对我们的洗礼，我们慢慢开始了理解，原来多模态在人工智能术语领域中，往往指的是感知信息，如图像、文本、语音、视频等。那么我们将这个术语挪用到文娱行业中，毫无疑问指的就是图书、漫画、动画、影视、音乐、社交互动等多种形态的产品。

# 

# 而我们认为，随着AI技术的发展，文娱行业一定会进入到一场高烈度、多维度、超位面的多模态战争。也就是说：

# 

- # 本来做网络小说的公司，开始自己做漫画；
- 本来做漫画的公司，开始自己做动态漫或者动画；
- 本来做影视的公司，开始收购⼀家文学公司做条漫；
- 本来做音频的公司，开始做童书出版……

本来只能做模态A的公司切入模态B；本来做模态B的切入模态A……每⼀家有AI能力的IP公司，突然都说“我全都要”。

![](https://mmbiz.qpic.cn/mmbiz_png/UgtzVuzhFd5COeWGy9o07bERnkPCj6kAzmAQebCfbqemG6ACXiawRljx3cyd3QQo0ia2YtpWQXMs37RkkQrGAgicg/640?wx_fmt=png)

但我们认为，这毫无疑问是一件好事，就像郭帆导演前面所说：“我们和好莱坞在同一个起跑线上。所以是有可能会有弯道超车的机会的”。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7tMuAmiadsyvAk4l30TZvmgP03RGX0iaosuL5yVawsdblYqeWUcOTHYoQ/640?wx_fmt=png)**现状：无法沉淀IP**

我们先聊聊现在中国娱乐行业的一些问题吧：

![](https://mmbiz.qpic.cn/mmbiz_png/UgtzVuzhFd5COeWGy9o07bERnkPCj6kAYpdVPQJSDgpQKaKkAvEgQUf3XazYDeWukEQvT11NDUtzFxKOq3ibH1g/640?wx_fmt=png)

因为前几年行业的混乱和盲动，让业外资本受伤了，所以回看AI概念出现之前的文娱行业，你会发现是典型的业务估值逻辑，大家在严格拆分这个公司的业务面，以PE估值法界定，没有给予任何想象力的溢价（虽然仍有些中国A股特色的高估值）。

但海外基本上大公司只有⼀两种逻辑：那就是严格的媒体公司、IP平台，当然绝大多数大公司是两者兼有。这里面最典型的无外乎就是迪士尼，图片出版、流媒体、电影、电视网、乐园、特许消费品，一个成功IP少说也得经过七八种模态，几十年的发展……（《小美人鱼》除外![](https://res.wx.qq.com/t/wx_fed/we-emoji/res/v1.3.10/assets/Expression/Expression_21@2x.png)）

![](https://mmbiz.qpic.cn/mmbiz_png/UgtzVuzhFd5COeWGy9o07bERnkPCj6kA9vT7fczheuibu6uWGwotfuriaFqSbQ7SPn3GibXyTokQpRgIWGow3qn9A/640?wx_fmt=png)

十年过去了，从娱乐资本论成立开始，大家就说要做中国的迪士尼，那为什么中国没有一家做成迪士尼呢？         

- 国内文娱行业全产业链公司发育不够完善，往往需要集中精⼒单点突破；
- 某种模态需要深耕，比如说制作动画，需要复杂的工序，高的技术含量，造成高准入门槛。
- 在急功近利的大环境下，大家更愿意做安全性的公有IP，而很少敢于持续性投入资源。比如说动画电影领域，更多地还是孙悟空、哪吒这样的，很少有像《熊出没》这样持续耕耘自己的IP
- 结果⼀个IP的多次开发往往需要不同的公司反复介入，有时还会造成IP的浪费。

所以在传统模式下，只有特别发达的产业和头部公司，才有可能做成围绕⾃有IP，在自己的集团内部完成跨模态的开发。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7ejDSZf97dAE3mMYqSpwDp0blV0YsOONibSOjLz8EycRV8uxj7xc8QIg/640?wx_fmt=png)**机遇：AIGC技术大发展**

幸好，我们遇到了人工智能图形创作（AIGC），有望彻底改变娱乐业，通过优化制作流程和降低成本。

1.提高效率 ：王长田说，AI可以提升制作动画的效率⾼达30%，从而实现更高产出。

2.扩大可能性 ：AI可以用于动漫可视化、图形创作和⾃动配⾳。

![](https://mmbiz.qpic.cn/mmbiz_png/UgtzVuzhFd5COeWGy9o07bERnkPCj6kA8aib1sXRf8Yl1NPLH2grHH6RaC0NL0lVAadQjJVTyicLToWicfFbLGaag/640?wx_fmt=png)

当然毫无疑问现状都是在一个不成熟的状态，但我们在探访了30多家AI公司之后，我们认为现在发展的时间线已经近在眼前：

- 目前AI已经能做到简单的故事架构设计、美术概念设计、真人动漫风格化（类似《蜘蛛侠：纵横宇宙》的多风格）
- 预测未来1年内，可控的美术设计将会成熟
- 预测未来2年，AI生成视频就会具备⼀定的精度和可控性

而随着这些技术的发展，在未来，我们认为：

- 先是《石头剪刀布》这样十余人团队，一个月制作了7分多钟的动画电影，惊艳全场。
- 随着技术升级，⼀个几人小团队，只花7、800万，就做出⼀个70分水平的东西，占领了原本上百⼈花费几千万才能占领的市场（这是一个大公司老板亲口对我说的）
- 漫画、条漫、动画番剧、真人短剧，随着制作成本的降低，将呈现出上十倍的供给量，全行业的竞争烈度加剧。
- 同时传统IP，可以更好的通过衍生小说、二创、动漫、真人剧、互动游戏等多种模态横向扩展。
- 就像现在付费小说和免费小说市场的分化一样，付费小说的人群维持在千万人左右的稳定市场，而免费小说在人数上数倍于这个市场，但成熟的付费市场和有效的改编市场，则由头部的IP决定，在这其中，辰东、会说话的肘子、唐家三少、天蚕土豆、猫腻等仍是富豪榜上的常客，他们的新老IP靠着动画化、影视化和二创化，反而在赚钱效率上要远超其他新人新IP。
- 也就是说，随着供给量百倍千倍的提升，反而让原有的IP用户认知效应更为放大——掌握IP者得天下，掌握受众者得天下

就像我们的嘉宾，中国作协科幻文学委员会副主任陈楸帆所说：“在多模态的基础上，我相信会有很多的机会出现，但最后也可能会变成一个大鱼通吃的结果，无论什么样的结果，它一定是一个非常激烈的竞争。”

![](https://mmbiz.qpic.cn/mmbiz_jpg/UgtzVuzhFd5COeWGy9o07bERnkPCj6kAORgNROjwItqezypxiah0UR5MOMYCAJQnBY5x4HaqqLbLxCsWYWFXj0g/640?wx_fmt=jpeg)

当然，我们也衷心希望像陈老师所说的——“当AI生成的内容，大规模的出现在市场上、迅速的占领市场，其实它会带来审美上的疲劳，有可能会发生人类重新回过头来觉得我们还是需要人，还需要一些属于人的生命情感的东西——所以它对于我们人类的创作，是一个螺旋上升式的曲线。”

另外在这里，我们也小小的做一下广告，也是我们在3个月内用AI优化自身工作流的一个体会：

- 成立AI培训部门，针对外部培训和企业内训，即将和某教培机构合作开课
- 保留必要的图片合作外，争取以后每个采编都能自己生图、用图
- 本次论坛及上影节报道的各环节速录类稿件，均由AI生成
- 我们自己用AI为某项目做了⼀个动画片头

但在最后，我们需要明确：

①由AI生成”并不意味着人类可以“偷工减料” ，从最终结果上看，我们报道的质量和标准和以前没有区别。

②AI不是取代人类，而是解放人类。我们的记者和编辑对AI⽣成的结果负最终责任，而他们也拥有这些作品的原有权益

谢谢大家！
