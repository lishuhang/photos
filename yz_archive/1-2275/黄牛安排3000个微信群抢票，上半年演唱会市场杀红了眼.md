# 黄牛安排3000个微信群抢票，上半年演唱会市场杀红了眼

作为一个现场音乐爱好者，你最近有没有为看哪场演唱会而感到纠结？

继李荣浩和老狼之后，五月天演唱会火力全开。从北京市文旅局4月3日签发的演出许可来看，5月26日到6月3日，不到10天时间，五月天要在鸟巢连续举办两轮演唱会。这两轮演唱会时间相隔仅3天，一共演出6场，涉及56首歌曲。

几乎同时，北京草莓音乐节也公布了在北京世园公园从4月29日到5月1日的具体演出议程。有网友称，歌迷真的是过年了。

接下来，包括周杰伦、薛之谦、张杰、张韶涵、刘若英等一系列明星都将开启自己的全国巡演。根据媒体不完全统计，今年明星演唱会至少200场。

“没办法，市场需求在这里。之前定的演唱会一拖再拖，三年了，粉丝们都憋坏了，我们也想抓住这波报复性消费的机会，演唱会开的越早，在人气上越有优势。”一家演出商机构表示。

面对饥渴的粉丝们，明星艺人、演出商、黄牛，全部开足马力向前冲。

娱乐资本论了解到，有黄牛动用3000多个微信群一起抢票，周杰伦内场票价格飙升10倍；二线明星演唱会门票是票面价格两倍起步；观众买完又“被退票”的情况比往年更普遍；甚至火爆的演唱会市场还催生了中国新一代黄牛。

可以说，整个演唱会市场一片繁荣，又一片混乱。

有业内人士担忧，今年演唱会市场过于火爆，在消化之前存货的同时，也在透支粉丝们的消费能力。受消费大环境影响，这种火爆的情况不会持续太久，今年集中放量会后，明年市场将恢复至稳定期。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)**半个月前，黄牛越卖票越亏钱**

三年了，粉丝们终于去现场看了明星演唱会。

3月25日晚上，当李荣浩开始翻唱《麒麟》的时候，全场沸腾。有一位五音不全的女粉丝全程跟唱《喜剧之王》，被周围观众拍下来发网上。视频评论区有人表示，980才买到的看台票，这位小姐姐现场唱了900，李荣浩唱了80，跑调并不重要，重要的是发泄，是疯狂。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

如果单从一级市场的卖票和票面价格上看，其实很难感受到今年演唱会市场的火爆。毕竟，头部明星演唱会开票即售空是常态。根据摩天轮票务CEO 刘斅的观察，当下的明星演唱会票价与2019年相比并无大变化，一般还是380、580、780、980、1280、1580等几个档次。

但真实的市场和门票交易情况，只有黄牛们才知道。

“今年演唱会市场火爆到超乎我们的预期，让我们措手不及。”一位大黄牛表示，大概半个月之前，只要卖票就会亏钱，不卖才能止损。原因是市场太火爆，所有演唱会票价时时在变，一直上涨。

黄牛手中一般只有10%的现票，其余90%都是有观众购买的时候，临时从同行手里调货。今年的情况是，他们上一秒刚挂在二手票务网站上一个价格，下一秒找到同行时，同行的售价比自己的标价还要高200多块。

“这种情况下，就没法按照票务网站上的价格卖给观众了，我们会把观众的定金或者是全票价格再直接退回去，就相当于我们违约。不然卖得越多，亏得越多。”

不久前，网上还流传一张微博截图，上面显示，自己做票务市场7年，都没有见过今年这样的怪现象。像李荣浩这样的二线明星，不仅票价一开场票价就得加几百，票价直逼一线明星。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

一位广东的演出商认为，李荣浩虽然是二线明星，但今年他过去三年在社交媒体上积累了很高的人气，因此演唱会门票抢手也很正常。

李荣浩最近的《乌梅子酱》风靡短视频平台。“李荣浩的看台票能卖到1200多。”

高志从2017年开始做黄牛，慢慢做大，成为黄牛核心层的一员。现在高志手上的SKU很多，包括演唱会门票、茅台、潮玩等，几乎什么火抢什么。

“为了抢周杰伦的演唱会门票，我发动3000多个微信群一起抢。我会给里面的成员培训，教他们抢票。他们抢到票了之后，我不但返还给他们佣金，还承诺，今后他们要买的演唱会门票，我以原价出售给他们。这样他们就抢得很积极。”

高志表示，这两天，演唱会票价浮动没有那么频繁了，也敢卖了，自己筹集了300多万用来在各场演唱会之间调货。

“中国第一代黄牛其实是bigbang演唱会那时候诞生的，后来第二代黄牛是抢小米手机的那些人。过去三年，几乎没有大型演唱会，很多以前的演唱会黄牛都不干了，今年一下子这么火爆，又进来了一些新的黄牛。他们很可能就是中国第三代黄牛。”

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)**谁是演唱会背后拿镰刀的人？**

市场需求是今年演唱会爆火的原因之一。上述黄牛表示，跟整个大消费环境整体低迷的趋势相反，粉丝们的购买力非常强，不怕卖高价，就怕没票。“内地有些粉丝花上万元买票去香港看林俊杰的演唱会。”

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

除此之外，演唱会主办方以及演出商等整个演出链条上的集体推动，急于赚钱，也是整个演出市场火爆的重要原因。

过去三年，受疫情影响，演出市场尤其是明星演唱会，几乎是零演出，颗粒无收。不少演出商由于接不到业务，已经倒闭或者转行。

“好不容易市场恢复了，活着的演出公司需要赶紧开项目，赚钱，才能保住公司。”西安DAYS大圣创演空间主理人文路认为，从某种意义上来说，今年办一场演唱会是之前1.5倍甚至是2倍的成本。

大型明星演唱会一般是前半年就预订场馆，安排巡演路线。如果放鸽子的话，意味着之前做的一切筹备投入都打水漂了。之前有几位明星已经官宣要办演唱会，但都因为疫情耽误了。今年的演唱会自然也要把打水漂的成本补回来。

资金压力不仅仅来自于已经发生的成本，还来自对未来经济走势的预判。

从整个消费趋势来看，今年下半年甚至明年的经济形势都不甚明朗。各家消费品牌的营销预算也都集中在今年Q1，并从Q1到Q4依次递减。演出主办方们也是这样想的。办得越早，越有主动优势。

对于核心粉丝来说，不管票价多贵，不管在哪个城市，自己爱豆的演唱会一定要去看。但对于大部分想要去现场寻找情绪价值的路人粉来说，他们每年花在演唱会上的预算有限，而且，也没有固定喜欢的明星。

这样一来，谁的演唱会开得早，他们就会Pick哪一场，时间越往后拖，路人粉的兴趣就越小，票价和演出观众数量都难以保证。“尽快把观众们口袋里的钱搞到手，落袋为安方为上策。”一家二手票务公司中层表示。

快速回本的意图与一部分高企的演唱会成本叠加在一起，就全部都转嫁到了票价上。

如果按照大麦、猫眼等平台上公布的票面价格出售，一场演唱会的收入可能很难覆盖明星出场费，更赚不了几个钱。为了获得更高的利润，一少部分平价票给到正规票务平台后，其余的票都通过各种渠道，以溢价的方式转到了黄牛手里。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

“主办方为了赚钱，会通过一个中间地带的人，把一部分票溢价后直接转给黄牛。这个人一般不露面，但是深受演唱会主办方信任，是主办方与一级黄牛代理之间的防火墙。”上述黄牛表示。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)**线下演出市场的火爆还能持续多久？**

毫无疑问，不管是数量上还是票价上，今年都是线下演出的大年。

一位高频观演的年轻人表示，北京一场线下脱口秀价格都880元，一场livehouse演出的票价都飙升至1000多，开心麻花的舞台剧，年前票价380，同样的位置，年后就变成580了；上海一场交响乐团音乐会搭售周边一起卖票，价格高达1999元。

但任何事情都有AB面，火爆的线下演出的背后也有层层隐忧。尤其是演唱会市场，有行业内人士认为，当下演唱会的爆火在一定程度上会透支消费市场市场。

前述广东演出商表示，今年个人演唱会还是头部明星的主场，比如周杰伦、林俊杰、薛之谦、华晨宇等，但除此之外，一些新人也趁市场行情好来凑热闹，这样可能会“伤到”自己。

“头部歌手在作品上是有一定积累的，他们发过的作品可以撑得起来多场次的巡演。但是，新歌手作品没那么多，如果强行开多场演唱会，这样很难保证演唱会后几场的质量，也是对粉丝的不负责，很可能会引起脱粉。”

事实上，对于中小演出公司而言，今年也不是大踏步前进的好机会。因为演唱会市场出现一个新趋势，垄断性增强。

一些头部明星演唱会都有明星背后的经纪公司亲自操盘，它们与演出商的合作模式分为大包和小包。大包指的是，市面上资本实力强的演出公司主动与艺人经纪公司谈合作，要求演出多少场，并包下所有费用与落地执行；而小包指的是，艺人经纪公司找到各地的演出商分别合作单场演唱会。

过去几年，市面上能做大包的演出公司只有少数几家。它们可以一次性拿下头部明星的全国巡演场次，然后跟各地的头部演出商合作，小型演出商很难参与。

在聊天的过程中，有演出商表示，几年前，他们帮李荣浩做过一场二线城市的演唱会。当时整个盘子一共是500多万，但是票卖的并不好，最后只卖出去20%。无奈，这家演出商只得以场地有问题为名义毁约，赔偿违约金。

“这样一来，我就只赔偿前期的150万定金就可以了。不然票房收不回成本，强行开演的话，我们得赔四五百万。今年是不行了，高攀不起了，像我们这种级别的演唱会公司是拿不到李荣浩演出的。”

这家演出商表示，中小演出商今年其实应该谨慎一些，头部明星演唱会神仙打架，留给小明星演唱会的市场份额并不多，一旦操作不慎，可能就很难回本。“可能需要到明年，中小演出公司才能再试试机会，今年机会不大。”

在明星演唱会之外，线下演出行业更大的隐忧来自音乐节。

今年成都某些月份一个月就有6-7场线下音乐节，相比之下，像北京这样的一线城市一年大概有10-12场，平均至单月，每月约2场。明显看出，今年成都音乐节供给过于饱和。而当下音乐节同质化严重，观众看完一两场就知道大概阵容，区别只在于城市的不同。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

“我们预测，今年下半年的音乐节市场会出现下跌的可能，要想脱颖而出，必须保证作品品质。所以，音乐节暂未列入我们的合作计划中。”文路表示。
