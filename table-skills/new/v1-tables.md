## Table 2: AI招聘动机分类（象限图/分类矩阵图）

**数据：**

| 公司类型 | 核心压力 | AI优先进入环节 | 代表公司 |
|---------|---------|--------------|---------|
| 降本型影视公司 | 制作周期长、成本高、爆款不确定 | 分镜、概念设计、AIGC物料、AI漫剧 | 华策、正午阳光、启蒙影业 |
| 增效型影视公司 | 需要寻找新内容增量 | AI产品、AI漫剧、虚拟偶像 | 博纳、恒星引力、欢娱、耀客 |
| 出海型影视公司 | 海外本地化和社媒运营成本高 | 多语种物料、短视频切条、海外AIGC运营 | 柠萌 |
| 经纪公司 | 真人艺人成本和风险高、粉丝内容供给压力大 | AI艺人、虚拟偶像、AI粉丝运营 | 乐华、哇唧唧哇 |
| MCN/直播电商 | 人力密集、ROI压力强 | 场控、客服、审核、运营、短剧生产 | 无忧、遥望 |

**图表类型：** 分类卡片矩阵 / 分层条形图（5 个类别，每个类别展示三类信息）

**动画描述：**
- 5 个分类区块依次从下方滑入并淡入（stagger 200ms）
- 每个区块先显示类型名称，然后展开显示核心压力、AI环节、代表公司
- 代表公司以标签（tag/badge）形式弹出
- 标题："AI招聘动机分类"

**配色：** 每个类型一种主题色
- 降本型：#FF6B6B（红）
- 增效型：#FFA940（橙）
- 出海型：#00D4AA（绿）
- 经纪公司：#7B68EE（紫）
- MCN：#E040FB（品红）

## Table 6: 核心洞察汇总（信息图卡片）

**数据（6 条核心洞察）：**

1. 半数岗位点名要AI → 13家公司 41个AI岗，传统影视30个
2. 华策让人用AI，博纳让技术做AI → 两种截然不同的路径
3. 恒星引力卡985学历 → 梭哈AIGC姿态
4. 乐华"纯AI生成、无真人演员" → AI改造偶像的最激进尝试
5. 耀客签约AI数字演员 → 仿真人短剧分成系数高达60
6. 遥望为AI短剧专招HRBP → 兵马未动粮草先行

**图表类型：** 信息图卡片轮播 / 时间线式排列

**动画描述：**
- 每张卡片依次从右滑入（或从下弹入），停留 2 秒
- 卡片内数字做计数动画，文字淡入
- 编号（1-6）以大号字体显示
- 标题："核心洞察"

**配色：** 每张卡片用不同主题色点缀

## 通用规格

- **比例：** 9:16 竖屏（450×800px）
- **内容区：** 画面正中 3:4 或 1:1 区域，四周留边
- **字体：** 统一阿里巴巴普惠体（5种字重：Thin/Regular/Medium/Bold/Black）
- **背景：** 深色 #0a0e27
- **动效：** 打开页面自动播放，播放完毕后右下角显示 Replay 按钮
- **Replay 按钮：** 竖屏时位于画布右下角内侧；宽屏时位于画布外右侧（方便录屏时裁掉）
- **数据来源标注：** 每张图底部小字 "数据来源：娱乐资本论统计"
