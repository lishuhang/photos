# 沉淀IP新价值，腾讯视频特色“长期主义”有新主张

随着互联网视听行业的飞速发展，到近两三年去泡沫化的阶段性完结，在大环境影响也已经逐渐消散之后，长视频行业自去年下半年明显进入了稳步发展的新阶段。

随着2022年人均GDP超过8.5万元，用户娱乐消费需求不断升级，此刻同时传来利好消息：长视频行业在经历十年探索、发展与财务亏损后，通过2021年至2022年的全行业降本增效行动，带来了理性市场的回归。**腾讯视频接近盈亏平衡点，有的平台开始盈利，这或许可以成为长视频行业从第一个“十年”跨入下一个“十年”的分水岭。**

其实洞察长视频行业的过往十年，不难总结出腾讯视频独特的“长期主义”战略：即**以长线IP开发为内容积淀、构建资源池，并通过持续耕耘助力行业的不断增长。**

在今日（3月30日）的第十届中国网络视听大会上，腾讯公司副总裁、腾讯在线视频首席执行官孙忠怀也提出了，腾讯视频将在长视频行业“新十年”中，坚定不移地践行该项特色“长期主义”。

作为行业的主要参与者，腾讯视频见证了中国视听产业的发展与变迁，以及行业秩序不断规范化和清晰化的过程。此番演讲也重在讨论长视频行业在进入稳定期后，该如何理性面对下一个“十年”，如何加强科技、尊重创作，以保证输出更多的好故事。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)**长线IP增值、创新内容格局，腾讯视频即将“加速”**

自2021年下半年开始，长视频市场变了，这是整个行业的从业者可以明显感知到的。用户对头部精品的极致需求导致供需关系变化，为满足用户追剧的多样化、创新类型不断迭代，新赛道崛起，但同时平台竞争进入存量时代。

这一切的变化督促行业做出了降本增效的决定，并且很快在一年半的时间内看见了显著的成效，腾讯视频则是不断做出喜人成绩，在行业里脱颖而出。

回溯2021年开始的腾讯视频，头部白马型内容与小而美、小正大的黑马内容频频交叉出现。2022年开年的轻科幻爆款剧《开端》，和2023年开年的科幻爆款巨制《三体》，即带有好内容的传承与视听创作升级的重要意义。期间诸如《梦华录》《星汉灿烂·月升沧海》《昆仑神宫》等爆款热剧，也从不同赛道满足了全民的娱乐供需。另外，《御赐小仵作》《爱很美味》《听说你喜欢我》在各自的垂类赛道中也相继出彩。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

同时，综艺赛道里《脱口秀大会》《半熟恋人》《令人心动的Offer》《心动的信号》稳固腾讯视频在喜剧、脱口秀与生活职场情感等类型领域的明显优势；《斗罗大陆》《斗破苍穹》《完美世界》《吞噬星空》等年番作品，在动漫领域不断增加用户粘性，**腾讯视频不仅凭此验证了年番动漫的长期价值，也打出了相比其他平台更为个性化、长期化的差异打法。**

放眼内容市场，近两年从圈层热剧到国民爆款，可以说是百花齐放。**长视频行业的新局面已然打开。**

**IP资产素来是腾讯视频打造差异化竞争力的核心要素，也是平台保持长久生命力的根基。**在互联网影视的第一个十年里，腾讯视频已经储备了5-10年的优质IP，并且以此培育出相当数量的好内容呈现给用户。例如腾讯视频从《鬼吹灯》IP系列第一部作品《精绝古城》诞生至今已经有超过六年之久，口碑不断攀升，今年以及未来还将继续推出新作。

**下一个“十年”，腾讯视频仍旧有多项优质IP为平台在全行业“长期主义”的洪流中打出明确的差异化而保驾护航。**继《鬼吹灯》IP系列之后，《三体》第一部电视剧完结之际公布将要开发《三体II·黑暗森林》与《三体·大史》，该剧豆瓣分已经涨至8.7。2019年初试《全职高手》剧漫联动后，《斗罗大陆》将贯彻此做法进行长期开发，动漫赛道作为腾讯视频继电视剧之外的第二增长曲线，也将成为平台打造IP孵化差异化的一大优势。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

除此之外，将要开拍的爆款《庆余年》续作，2019年宣布拿下的古龙全IP版权，以及五大名导携手各大知名编剧打造、正在拍摄中的《金庸武侠世界》系列五部作品，成为了腾讯视频稳固武侠题材赛道的底气。根据头部奇幻小说作家尾鱼打造的“尾鱼奇幻宇宙”五部作品，同样在开发拍摄阶段已经备受关注，被视为奇幻赛道未来的重头白马项目。

作为综合型平台，腾讯视频打造的《脱口秀大会》系列为用户带来的欢笑、为行业输送的人才，以及《风味人间》系列为首的美食纪录片，也在长期的孵化中从分众走向大众。

**淘汰低效产能、追求长效价值，一直是用户、平台和市场的多方共识。**

而长效价值中，不仅IP的开发沉淀增值，二十大的号召和国家广播电视总局的指引下，平台所肩负的社会责任感也是长期战略的感性核心价值观。为此，腾讯视频近三年推出大量优质主旋律作品，《山海情》《大江大河》《县委大院》《功勋》等作品实现了口碑与流量的双收，《埃博拉前线》《狮子山下的故事》也获得了用户认可与热播。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

**下一个“十年”，腾讯视频的“长期主义”还将继续以讲好中国故事为重点工作，推动中国故事不断出海，努力早日实现中国的文化强国梦。**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)**视听升级、尊重创作，艺术与科技双驱牵引**

**通过提质增效的快速迭代，互联网影视精品化已经进入4.0的进阶阶段。**内容的优劣在感性认知的创作中，将加入更多时代洪潮发展下的科技元素，以保证视听质量与用户体验感能够同步升级，**这也是为腾讯视频可以在新阶段打出差异化特色的“长期主义”做准备。**

最为直观的是影视领域在拍摄过程中的视效技术进化，腾讯视频早在2019年热播的《精英律师》中就以LED特效数字屏幕替代传统绿幕拍摄技术，自此全行业的都市剧开始逐渐普及这一创新视效方式，不仅呈现效果避免反色、更为真实，还减少了大量剧组布景转场的时间精力，达到了技术方面的降本增效。

同年的《全职高手》中，线上世界部分总计45万帧画面，涉及岗位共50种，一共动用了566名工作人员，摆脱了传统后期特效动辄一天一帧的渲染周期，演员的动作、甚至灯光与场景能实现同步渲染，大大缩短了后期流程。

而在科幻真人剧《三体》的拍摄过程中，小说名场面“古筝行动”的复原呈现，也是通过在计算机里做动力学测试，突出船的重力感和扭曲的刺激感，模拟画面效果。原本难以想象的视觉画面，最终获得用户与书粉共同的大力赞叹。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

在腾讯视频另一重点的动漫领域，周播动漫《吞噬星空》作为科幻题材，没有实际建筑可以参考，大量设计完成了从零到一创造，场景平均模型数量超过1.8万个，总面超过1.2亿面，都是依靠多个工序环节的密切配合、高性能的软硬件服务和算力支持完成。

**2021年，腾讯视频即提出以艺术战略与科技战略创造内容美好价值。如今科技在不断跟进，“艺术”则是通过更加尊重创作者来实现。**

在中国影视的半工业化进程中，科技是激发创作思路的工具，创作是推动科技变革的原点，两者相辅相成、互相反哺，共同驱动长视频平台内容生态的进化，为用户提供更加多元化、高品质的娱乐内容，并且同时享受更为舒适、便捷的在线娱乐生态生活。

所以**在科技辅助下，坚定拥抱并尊重影视内容的“手艺人”，也是腾讯视频“长期主义”中重要的一环。**这里的“手艺人”包括从文本创作开始、到拍摄阶段的编剧、导演、演员、声优、原画师，以及其中服化道、舞美、后期视效等技术环节，每一项人力创作都值得尊重。

科技是理性的，优质内容的源头是创作端与制作链路中每一个环节的感性创作加码带来的具有个性化的活力，腾讯视频在未来会更加尊重创作与制作链路中的每一个生产环节，提升“手艺人”的行业地位，最终实现提高生产、优质内容的共赢。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)**腾讯视频还有多少助力好内容的“法宝”？**

**关于贯彻“艺术+科学”战略，腾讯视频另有诸多“法宝”助力其实现未来差异化特色的“长期主义”。**比如腾讯视频利用综合生态优势，通过开放合作、排播理念、IP衍生、线下营销等形式，不断助力好内容的诞生、出圈以及价值加码。

首先，为了持续提供精品内容、完善平台服务，给到用户与合作伙伴更多的信心，腾讯视频坚持视听产业应以合作共赢为共识，因此长视频内容创作方面，非常乐于与合作伙伴共担风险，也因此吸纳更多优秀团队共创高品质项目，这些都是为行业增加好内容的基石。

比如2016年发起的分账剧合作形式，至今已有7年时间，内容从野蛮生长到稳定精品输出，平台在其中给到的分账策略功不可没。

经历7年六次的策略升级，腾讯视频在2023年最新分账规则中引入“激励收入”概念，以替换之前的“保底”概念，为了让创作团队不是想着保底，而是想和平台共创优秀作品，腾讯视频同时将非独家合作的分成提升至与独家合作分成相同的100%，累计激励收入最高可达1亿元。

**激励好内容之后，促进好内容的传播效果也是维稳的长期之计。**腾讯视频今年升级概念，将口碑、质感向短剧集中排播达成集群效应，类型相同的小众剧集整合排播营销达成连续效应。

再延续到营销侧，2022年《梦华录》《星汉灿烂·月升沧海》《昆仑神宫》《爱的二八定律》四部剧初尝“大结局点映礼”，满足用户追剧的仪式感，变相增加会员权益，提高平台内外社群互动，增加用户粘性与沉浸式体验。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

当然，沉浸式体验不止可以来自于线上，最为有意思的是将用户体验延展到线下。《梦华录》在去年6月引发全民追剧热的时候，腾讯视频将此IP延展至线下场景，在长沙大悦城星河广场开展《梦华录·风雅梦话游》主题沉浸式集市，以剧集故事为背景，高度还原汴京的大宋风貌，一个月内即收获2万多现场体验人次和20多万的线上观看人次，成功加深线上用户与线下人群对《梦华录》这一IP及腾讯视频品牌的认知和印象。

基于《梦华录》的成功，腾讯视频也将于今夏为《斗罗大陆》的粉丝群体量身打造线下体验场景，在成都宽窄巷子举办集美术陈列、线下体验、新媒体演出、游乐、音乐会、美食于一体的沉浸式 IP 嘉年华。

**当一个优质IP获得了市场的肯定，必然可以获得更为长久的生命力，实现长线价值。**自流媒体产业掀起全球热潮之后，从传统的迪士尼到新兴流媒体平台已验证过。近几年腾讯视频孵化的IP项目市场认可度，再次证明这一做法的成功。
