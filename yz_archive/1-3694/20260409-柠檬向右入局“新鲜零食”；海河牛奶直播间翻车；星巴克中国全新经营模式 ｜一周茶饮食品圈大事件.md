# 市场热点

**01**

**柠檬向右入局“新鲜零食”**

近日，茶饮品牌“柠檬向右”，宣布入局“新鲜零食”赛道，旗下“向右新鲜零食店”首店计划于今年5月正式开业。柠檬向右将采用“独立店+融合店”并行的策略。融合店将依托现有茶饮门店实现“茶饮+零食”的场景联动，向右新鲜零食店采用标准化的空间方案，单店面积控制在100至200平米区间。在此之前，品牌已经完成了两次重要的战略试水：2025年实现从柠檬茶单一品类到全品类健康茶饮的跨越；此后又逐步切入轻咖啡赛道和早餐时段。此次入局新鲜零食赛道，正是其在“茶饮+”多品类融合路径上的又一次纵深推进。

**02**

**鸣鸣很忙CEO晏周亲抓新项目**

近日，市场人士称，鸣鸣很忙CEO晏周的朋友圈迎来了近期唯一更新的一张“招募产品公告”海报。据了解，晏周正在主导一个“新项目”，如果非要给这个新项目一个定语的话，那就是“全球建筑面积最大的零食城”、“线下优选版的糖酒会”。据了解，该店占地面积20000多平方米，负一负二层面积就超过12000平米，SKU数量将超过35000支。

**03**

**喜茶全国首家“蛋糕实验室”落地深圳**

4月6日，喜茶全国首家“蛋糕实验室”在深圳万象天地正式亮相。以「云上」作为空间主题，在提供经典茶饮同时，还推出8款人气产品及首发1款门店限定新品「爆芋泥木薯蛋糕」。本次是实验室板块cake lab首度走出heytea lab，在门店体系中以独立板块的形式与经典茶饮菜单融合呈现。此前，这一产品系列仅在喜茶全国3家heytea lab门店有售。

**04**

**胖东来梦之城项目一期建设规划公示**

4月7日，备受关注的许昌胖东来梦之城项目一期建设规划开始公示。依据规划，项目一期建筑面积46.7万平方米，3327个停车位。胖东来创始人于东来曾在社交平台发布信息称，梦之城是胖东来品牌成立以来投资规模最大的单体项目，项目建设和装修投资约65亿元，是一个超大型综合体，将涵盖百货、超市、电器、珠宝、电影、餐饮、茶叶、电玩以及酒店等业态。此次规划显示，梦之城项目一期主要建设商业综合体，建筑主体长299米，宽216米，高度50米，8层。一期之外，未来还将建设幸福讲堂、文化中心、酒店等主要部分。

**05**

**贾国龙押注“天边砂锅焖面”连开六店**

西贝副牌“天边砂锅焖面”此前陆续在北京、呼和浩特、上海、深圳开出5家门店，4月6日，全国第六店杭州黄龙万科店开业。除了呼和浩特，天边砂锅焖面落地城市均在一线城市的商场门店，这与西贝所在城市基本吻合。公开信息显示，西贝门店数量集中位于北京、上海、深圳、杭州等城市。另有其他门店工作人员表示，“预计是在4月10日开店，具体还得看情况。”

# 舆情热议

**06**

**老字号海河牛奶低俗直播间翻车**

4月7日，有媒体曝光，海河牛奶品牌直播间主播在带货过程中发表低俗言论，包含性暗示、性交易内容，观感极其不适。据了解，始创于1957年的海河品牌，在2024年商务部公布的“中华老字号”认定名单中成功上榜，是有着深厚积淀的国民品牌。涉事直播间的认证主体为天津市佰意电子商务有限责任公司，但直播间醒目挂着“海河牛奶”标识，主营销售海河全系列乳制品。目前，天津市佰意电子商务有限责任公司已发布致歉声明，称已第一时间交由专员严肃处理，将全面整顿直播规范与人员管理，坚决杜绝低俗违规行为，并已下架涉事主播相关视频。

**07**

**优思益幕后推手被查**

4月1日，浙江杭州滨江区市场监管部门对杭州索象营销策划有限公司开展了现场调查。当地市场监管部门对杭州索象与广州雅拉源之间业务关系等资料进行调查取证。调查发现，多位明星代言推荐的“澳洲优思益”叶黄素护眼产品，其澳大利亚地址是一家汽车维修厂，“进口”是包装的、品牌故事是编的、国际大奖是买的、专家网红是雇的。这款产品的营销策划是由杭州索象营销策划有限公司负责的。目前市场监管部门已对企业涉嫌广告违法行为立案调查。同时，对涉及的杭州社淘电子商务有限公司、杭州爱德思达传媒科技有限公司涉嫌广告违法行为也予以立案调查。

**08**

**胖东来继续硬刚王海**

4月5日，胖东来官方账号发布关于“王海测评”发布胖东来销售鲜鸡蛋检出角黄素的情况说明（二）：关注到“王海测评”发布的相关视频后，我们第一时间成立调查小组进行事件调查，将在售的正大、德青源、野迹鸣、黄天鹅、圣迪乐品牌鲜鸡蛋同时送往三家检测机构分别进行检测，由于检测项目较多，检测报告约150份，于4月3日全部检测完毕，重点检测了鲜鸡蛋重金属、兽药残留、添加剂、微生物等相关指标，检测结果全部合格，符合产品执行标准以及GB/T39438-2020《包装鸡蛋》、GB2749-2015《食品安全国家标准蛋与蛋制品》规定要求。同时，胖东来还表示，针对“王海测评”，将进行维权。

**09**

**霸王茶姬疑似被日本抄袭**

近日，有网友反映，霸王茶姬CHAGEE疑似被一家名为“CHARMING”的日本奶茶品牌抄袭，该品牌的LOGO和菜单都与霸王茶姬颇为相似，仅把logo中的京剧女性形象换成了和服女性形象。据报道，CHARMING品牌公开的电话号码是一个日本号码，并称将在大阪和东京开设两家门店。霸王茶姬客服就疑似被抄袭一事回应称，对于所说的问题非常重视，已将该问题详细记录。

**10**

**青海湖牌高原纯牛奶倒出透明苦水**

近日，有消费者称，近期购买青海湖牌高原纯牛奶开箱饮用时发现，牛奶盒内液体并非正常纯牛奶应有的乳白色，而是呈透明状，品尝后还有明显的苦味。连续打开4盒同批次牛奶，结果均出现同样的异常现象。3月31日凌晨，涉事企业青海雪峰牦牛乳业有限责任公司发布《关于“青海湖纯牛奶”出现个别异常问题处理的紧急通告》：购买20260120批次青海湖纯牛奶的消费者，可凭消费购货记录截图提交至热线客服，自主选择退货、退款或换货服务。同日，青海省共和县市场监督管理局发布《情况通报》：针对“青海雪峰牦牛乳业有限责任公司生产的高原纯牛奶产品包装内出现透明液体”的问题，共和县市场监督管理局高度重视，第一时间成立调查组，进驻生产企业开展全面调查，同步启动同批次产品召回封存工作，对企业纯牛奶系列备样产品抽样送检，并对全类别产品开展全面排查。

**11**

**今麦郎不再使用“手打”商标**

日前，媒体曝光今麦郎“手打挂面”中的“手打”实际上仅为商标，挂面为机器生产。4月2日，今麦郎集团创始人、董事长范现国发布视频称，“手打”商标是2006年注册，已经用了20年。为了避免给大家带来产品表述的困扰，决定不再使用“手打”商标。“我们决定不再使用‘手打’商标，从4月2日凌晨起，我们会立即停止生产所有带有‘手打’商标的产品。

**12**

**北冰洋被质疑NFC果汁添加量仅0.005%**

近日，有网友发帖质疑北冰洋一款标注添加NFC果汁的葡萄复合果汁NFC含量低。北冰洋葡萄复合果汁，净含量为125毫升，配料表显示NFC葡萄汁添加量为0.005%，除NFC葡萄汁之外，配料还包含有水、浓缩红葡萄汁、浓缩苹果汁。按照净含量换算，NFC葡萄汁仅有0.00625毫升，而一滴水大约是0.05毫升。北冰洋食品公司工作人员回应称，包装为如实标注，不存在错误，会将消费者意见反馈处理。北京市大兴区市场监督管理局表示，已收到相关线索，将依据证据开展调查取证并依法处置。

# 财报业绩

**13**

**和府捞面2025年利润大涨**

近日，和府捞面公布2025年度业绩，去年整体收入达26.06亿元，门店总数扩张至632家，其中新增联营店87家，联营门店盈利店率保持在95%以上。2025年，和府捞面全力推进“到家”业务，构建“到店+到家”双轮驱动的增长新格局，外卖与线上零售两大业务齐头并进，成为拉动业绩增长的核心第二曲线，外卖业务收入同比增长120%，利润同比增长123%；线上零售业务利润增长率达330%

**14**

**盐津铺子2025年营收利润双涨**

4月8日，盐津铺子发布2025年业绩报告。在2025年实现营收57.6亿元，同比增长8.6%，归母净利润达7.5亿元，同比增长16.9%。扣非归母净利润为7.1亿元，同比大幅增长25.9%。第四季度的营收和利润出现短期承压，Q4营收13.4亿元，同比下降7.5%，归母净利润1.4亿元，同比下降2.1%。公司分红率高达51.0%，显示出对股东回报的重视。

**15**

**燕京啤酒发布2026年第一季度业绩预告**

4月2日，燕京啤酒发布2026年第一季度业绩预告。预计2026年第一季度净利润2.5618亿元至2.7270亿元，同比增长55%至65%；扣非净利润2.5253亿元至2.6784亿元，同比增长65%至75%。据介绍，期内燕京U8产品继续保持双位数的高增长，为全年业绩目标的实现奠定了坚实的基础。

**16**

**妙可蓝多发布第一季度业绩快报**

4月3日，妙可蓝多发布的2026年第一季度业绩快报显示，一季度实现营业总收入约为16.26亿元，同比增长31.81%；期内，营业收入达16.26亿元，同比增长31.81%，C 端、B 端业务同比均有超 20%增长；剔除股权激励费用摊销影响后，预计实现归属于上市公司股东的扣除非经常性损益的净利润7229.89万元，同比增长11.33%。妙可蓝多在面向企业的B端业务持续发力，主要为餐饮、大客户增长驱动。

**17**

**5100藏冰川啤酒业务营收1.69亿元**

3月31日，5100藏冰川有限公司公布年度业绩，2025年实现收入3.58亿元，同比增长58.5%；实现净利润8623.9万元，同比扭亏为盈；基本每股盈利为亏损1.83分，上年同期亏损12.92分。2025年，该公司水业务分部实现收入1.88亿元，同比增长115.3%；啤酒业务分部实现收入1.69亿元，同比增长23.2%。水业务总体销量同比增长48.3%，收入同比增长115.3%。

# 资本动作

**18**

**星巴克中国与博裕投资交易完成**

4月2日，星巴克咖啡公司宣布与博裕投资的战略合作已正式完成。这项交易落实了星巴克全球于2025年11月表达的合作意向，也再次彰显了公司对中国这一关键增长市场的坚定信心。根据协议条款，由博裕投资管理的基金现持有星巴克中国零售业务60%的股权，星巴克全球保留40%股权，并将继续作为星巴克品牌与知识产权的所有者和授权方，向新成立的合资企业进行授权。在中国内地市场的约8000家星巴克直营门店将转为特许经营模式，由合资企业运营管理。

**19**

**达能有望收购澳洲乳企Made Group**

4月1日，澳媒引述“业内消息”称，德太投资正出售旗下澳洲食品饮料集团Made Group，交易估值超过15亿美元，达能极有可能成为潜在买家。据了解，达能曾在2020年参与竞购，但最终因时任首席执行官范易谋需要应对激进投资者而未能成功。本次收购若达成，将为其带来澳大利亚快速增长的健康消费市场机会，涵盖高端乳制品及植物基品牌。不过，据称来自中国的竞标方仍存在不确定性，新希望携手欢乐家被指也在联合参与竞标。

# 人事变动

**20**

**安徽金种子酒任命新总经理**

4月1日，金种子酒公告，聘任谢金明为公司总经理。此前，2025年7月2日，华润啤酒老将何秀侠突然辞去所有职务，提前近5个月结束任期，金种子酒总经理职位一直空悬，直到谢金明任职。谢金明2018年6月至今历任界首市委常委、常务副市长，阜阳市建设投资控股集团有限公司党委副书记、总经理，安徽金种子集团有限公司党委书记、董事长，安徽金种子酒业股份有限公司董事长等职。
