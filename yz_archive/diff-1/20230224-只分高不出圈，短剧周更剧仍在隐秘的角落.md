# 遗憾的是，这个高分是在《立功·东北旧事》收官那天才获得的。猝不及防的好口碑，已经来不及挽救这部12集短剧。

# 

# 本来，这是一部颇有科恩犯罪风格的黑色幽默，接地气的东北90年代图鉴，又有中国地方奇谈的元素，是近期不可多得的佳作。

2023年以来，《回来的女儿》《平原上的摩西》《显微镜下的大明》等短剧集相继上线，即便品质不差，但都没成为爆款。

         

除了短剧集，国产剧另一种播出形式：周更剧，也鲜有能在受众形成讨论热潮的，比如正在芒果季风剧场播出的《今生也是第一次》，话题敏感、直指女性生育，本可以有制造话题的疯狂势头，为何却播得悄无声息？ 

       

短剧集、周更剧，不能为市场认可的问题究竟出在哪儿？

## **给注水剧脱水**

# 

# 追溯中国电视剧的发展，剧集长短似乎经历了一个轮回。

# 

# 最初因为制作水平受限，剧作平均不到10集，到了90年代，万人空巷的《渴望》一播就是50集，中国电视剧井喷式发展，再到后来，电视剧为了长而长，“连续剧”就这么被观众叫出来了。

# 

# 其中的原因，当然是资本推动——电视台买剧按集算，导致注水剧大批出现，八九十集的作品屡见不鲜，除了屈指可数的精品，相当多长剧让人头大。

         

# 这也是为什么，当视频网站可以倍速观看时，很多观众毫不犹豫就点击了倍速。2021年，中国网络视听节目服务协会发布的《2021中国网络视听发展研究报告》显示，近四成00后有倍速观看网络视频的习惯，其中原因，包括剧情节奏拖沓、避开不喜欢桥段等。

# 

# 图源：川视新闻

# 值得注意的是，爆款如《狂飙》，能完整整看完剧集的观众数量也值得考究。相当多观众坦承，自己要么是在热搜上看完了《狂飙》，要么就在办公室摸鱼倍速get剧情。

# 没必要的注水劣质剧，当然要毫不留情地脱水！

# 

# 2009年，时任广电总局副局长胡占凡，曾在年会表示限制电视剧长度的规定即将出炉：“以后电视剧如果超过三十集，搞不好就播不出去了。”

# 

# 但现实情况是，瘦身令只停留在口头，并没有作为强制政策。

# 

# 风口浪尖上，个别电视剧进行了临时缩减，但大部分片方不为所动。

# 

# 2020年2月，中国国家广播电视总局印发《关于进一步加强电视剧网络剧创作生产管理有关工作的通知》明文规定，提倡电视剧以及网络剧的拍摄制作不超过40集，鼓励30集以内的短剧制作。

         

# 

# 短剧终于被官方“正名”，由此，视频网站也大刀阔斧各自展开布局。

# 

# 2020年5月，爱奇艺将“奇悬疑剧场”升级为“迷雾剧场”，上线《十日游戏》。没到两个月，《隐秘的角落》就成为爆款。

# 

# 短短12集，不过两周，却掀起了当年的造梗狂欢。

# 

# 之后，优酷同步力推悬疑剧场；在细分的微短剧赛道，腾讯视频也发布了品牌“十分剧场”。

# 

# 可以说，“剧场”代表了一种全新的运营模式和招商手段，最典型的是通过话题营销，带动讨论。而据2021中国网络视听发展研究报告显示，2020年上新的网络剧中，单部集数30集以内的占比达83%，这是相当值得关注的数字。

## **周更剧并不新鲜**

# 

# 国产周播剧出现的时间更早。2011年底，湖南卫视就推出了周播剧场。

# 

# 囿于审查制度，跟国外边拍边播的模式不同，国内的周播剧，都是已经拍摄完成的成品剧。这种按日播模式制作的本质上是一种“伪周播剧”。

# 

# 但购剧方对于电视剧剪辑却拥有最终决定权，为了提高收视率，大结局能专门剪出一集，前情回顾凑半集，任性剪辑曾导致观众强烈吐槽。

# 

# 尽管如此，湖南卫视对于周播剧场的经营，可谓不遗余力。虽然名称有过变更，第一周播剧场、青春星期天、钻石独播剧场、青春进行时……

# 

# 但却也打造过爆款剧目：《花千骨》《轩辕剑之天之痕》 《旋风少女》《楚乔传》《古剑奇谭》 ，都出现在符合年青观众审美趣味的暑期档，恰逢其时地推动了口碑造势。

# 

# 与此相较，曾尝试进行周播剧编排的上海东方、北京卫视、江苏卫视，播出剧目几乎没有动静，寂寂无名，有几个剧场甚至于2020年左右停止。

# 2021年，芒果TV引入“季风计划”，采用每周2集、单集70分钟，共12集的周播模式。招商会给出的季风剧场定位相当霸气：通过关照现实、先锋视角的题材，引发观众的深度共鸣。力图以电影级短剧树立起行业标杆，撬动行业格局。

# 季风剧场首部作品，是秦昊出演的《猎狼者》，本想借着“张东升”的势头，看秦昊打破自己、创造新的收视奇迹，但因为剧作本身BUG太多，秦昊饰演的护林员犹如有了不死之身的主角光环，即便摄影画面再好看，都有点让人难以带入。

# 

# 接下来余男主演的《谎言真探》有类似的毛病，好演员也架不住浮夸的故事构建。

# 

# 到了季风第三部，《我在他乡挺好的》终于出圈。凭着对北漂生活段子式地聚焦情绪，这部剧着实火了一把。观众一边吐槽剧情不符合现实，一边又在剧作中寻求打工人的共鸣，骂着看着，12集，38天播出周期，给本片打出8.1的高分，正片有效播放市占率一直在第10-30名之间，算是圈层小爆款。

# 

# 有血有肉的现实题材似乎更容易让观众忽略，周更两集的煎熬。2022年9月，哔哩哔哩播放的《三悦有了新工作》，由一个悲观的女孩，串起了丰富多元、充满悲欢的不同人生，温情的人世间伴着可无限截屏的鸡汤台词，迅速出圈。

## **突围之路道阻且艰**

# 

# 一方面，倡导创作短剧集的初衷，是为了提高电视剧品质化。

# 

# 另一方面，短剧集、周更剧两种形式，是中国电视剧向国际接轨的方式。

# 

# 众所周知，在全球获得好口碑的剧集，都是短平快，像《最后生还者》《纸牌屋》《权力的游戏》等英美剧，平均每部10集。

# 根据北美Deadline报道，流媒体奈飞信仰自己的短剧理论，认为剧集超过30集，成本就会变得昂贵。“集数越短，越容易吸引观众。”

# 

# 吊诡的是，国内观众可以接受快节奏的欧美剧集，分析着剧情等周更。但对于国产的短剧集和周更，似乎没有什么耐心。

# 因为国产剧盗版渠道太丰富，导致很多观众，抱着“养肥了”再看的态度。

# 一些短剧集，即便制作精良，但如果没在播放时间发酵出好口碑，它就会与观众失之交臂。

# 

# 短剧集和周更剧，要想抓住观众，首先自身品质必须过硬。

# 值得注意的是，在影视题材中，悬疑类囿于类型技巧更精准、影像风格更具特色、辐射人群VIP比例高，天然在各剧种中脱颖而出。

# 反转、高潮迭起、细节表演、氛围营造、悬念设置……好剧应该让人愿意N刷细思极恐。

# 《无证之罪》《白夜追凶》《隐秘的角落》等，就通过丰富的视听语言、美学构建为角色演绎加持。而短剧集《白色月光》《回廊亭》等，则在剧作本身出现问题， BUG太多无法自圆。

# 如果剧作线索太多，就要考虑是否出现重复剪辑，比如周播剧《消失的孩子》《今生也是第一次》，虽然形式创新，用了多角度叙事，但重复出现的片段和画面，极为考验观众的耐心。

# 

# 在短剧集和周更剧中，剧作的任何瑕疵都会被放大。这反而提高了对剧作的要求。

# 同一个制作团队，《隐秘的角落》当年的投入，是《无证之罪》的三倍，光找孩子角色的人选，就花了6个月。

# 另一方面，视频网站在品牌化精准运营上，一定要打持久战。

# 

# 好剧作和优质厂牌能形成良好的助推关系，万年影业的《隐秘的角落》帮爱奇艺迷雾剧场打开知名度，《沉默的证人》等剧作的后续编排，也加深了观众的印象。《立功·东北旧事》靠着高口碑，几位创作者理应有更好的后续发展，而元素叠加的逻辑则应该继续推进。

# 当入围柏林电影节的《平原上的摩西》出现在迷雾剧场，观众对于这部文艺气息浓厚的生活流悬疑片差点不买账，因为它的气质实在太独特了。

# 但《立功·东北旧事》《平原上的摩西》都有播完后，观看人数和评分还在看涨的现象，这也从某种程度上引发思考：短剧集、周更剧运营平台，应该如何推广。

#

# 尤其对于短剧集来说，剧播完了，并不意味着营销的结束。

# 爱奇艺高级副总裁陈潇曾对娱乐资本论表示：短的内容在市场上传播宣传，时效性就会低，如果说只做一部，**别人都在宣传五十集的剧，我宣传一部十集的剧，但是我投入的宣传资金是别人的两倍，别人才有动力。**这个模式让大家不敢去做精品短剧。现在我们为迷雾剧场宣传，我们用六七部的资金来宣传迷雾剧场，声量就更大了。

# 比如说，《立功·东北旧事》能否像《少年歌行》那样，弄个《少年特别企划》之类的产品，以剧集+生活番（或短综艺）的形式交错更新？

# 对于短剧集、周更剧难爆，国内行业大环境明显还在一个提升期，创作环境、观念认知、市场营销方式上仍有更多的升级空间。
