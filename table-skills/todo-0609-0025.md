## 制作方法

### HTML动画转视频
1. 使用 **Playwright** (headless Chromium) 以 450×800 viewport、2x DPI 截取动画帧
2. 每个HTML页面以30fps逐帧截图（`page.screenshot()`）
3. 使用 **ffmpeg** 将帧序列转换为 MP4 视频片段
4. Table 6 有480帧（16秒），分4批编码后拼接

### 视频拼接方法
1. **Python/PIL** 生成标题卡、字幕卡图片（深色背景 #0a0e27）
2. 将 Word 文档图片缩放至 450×800 画布居中
3. 静态图片用 `ffmpeg -loop 1` 转为视频片段
4. 需要缓慢推进效果的用 `zoompan` 滤镜
5. 图表动画视频用 `-t` 裁剪到指定时长
6. 19个片段用 `ffmpeg concat` 拼接
7. 添加 WAV 音频轨道
8. 用 `subtitles` 滤镜烧录 SRT 字幕（Noto Sans SC, 白字黑边）

### 工具链
- **Playwright 1.60** + Chromium: HTML动画录制
- **ffmpeg 7.1**: 视频编码/拼接/字幕
- **Python 3 + PIL/Pillow**: 图片处理
- **Noto Sans SC**: 中文字幕渲染
