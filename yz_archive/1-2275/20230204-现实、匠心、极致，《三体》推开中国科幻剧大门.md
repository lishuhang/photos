# 现实、匠心、极致，《三体》推开中国科幻剧大门

2月3日晚，腾讯视频同步官宣七部储备科幻片单。

最受瞩目的《三体II：黑暗森林》打头阵，IP衍生番外剧集《三体：大史》紧随其后，另有星云奖最佳长篇《群星》、北大物理系博士付强创作的《时间深渊》以及三部原创剧集《元都》、《二进制》、《完美之躯》蓄势待发。

科幻剧集是世界顶级流媒体平台的“压舱石”，Apple TV+的开局之作便是阿西莫夫的《基地》，家大业大的Netflix每年必有科幻剧集上线。但科幻影视作品的生产要求极高，一方面，它是对一国影视工业化能力的大考，**另一方面，科幻建立在科学之上，唯有科学与工业全球的领先的国家才具备生产科幻内容，特别是硬科幻题材的底气。**

而能让与Netflix“同题竞争”的腾讯视频有如此野心和信心的，正是开播即好评如潮的《三体》剧集。

从用户的评价来看，如今豆瓣评分高达8.1的《三体》剧集埋设了三个炸点——精确到场景和台词的质感还原，插叙闪回、交代全剧前因的年代戏，以及极为惊艳的超大特效场面，而这三个炸点又指向一个目标——**忠于原著，满足书迷。**

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bkfZicaJE5YJgciaiaf3m7lGP5n20YUT20O1mNfFXgdR4x8ibiaEibKuqaFyoA/640?wx_fmt=png)

娱乐资本论了解到，在前期筹备过程中，曾经有更长集数的版本，尽管更长的剧集意味着更大的商业化容量，但腾讯视频和制作团队最终选择了后者，并用长达五年的筹备周期，奉上了一部诚意之作。

五年间，中国用户的在线视频消费习惯迭代进化，破亿的腾讯视频会员基本盘，广告主和平台共同塑造的流媒体商业模型，以及随着《三体》知名度不断提升而带来的更多核心受众，都让打磨至臻的《三体》剧集，逐步发酵出其应得的播放成绩与用户口碑。

**“我们这行常说‘一剧一命’，对于同类的作品来说，命运可能会有天壤之别。”**

不久前，娱乐资本论见到了腾讯公司副总裁、腾讯在线视频首席执行官、《三体》总出品人孙忠怀，采访前一晚，《三体》剧集第15集刚刚播出，“人列计算机”的名场面得到完美复现，他告诉小娱，**“有朋友发微信告诉我，他‘跪着’看了一整集”。**

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bkxk2bic1Tsss4CmaqmR7HiaO9XXnThXyJ0avYcaJZMxroibf2UIohe2jGA/640?wx_fmt=png)

《三体》IP开发浮沉多年，终于等来了口碑爆发的一刻，这也是孙忠怀所说的“命”。

时间拨回2016年，彼时中国市场硬科幻大制作剧集的数量为零，但作为重度科幻迷，孙忠怀力排众议选择拿下《三体》剧集开发权，“总觉得如果我们不开发，交给别人，不太甘心”。到了2023年春节，在腾讯视频与合作伙伴的共同努力下，中国科幻剧集终于迎来从零到一的突破。

**“《三体》播出，大致就可以确定科幻赛道算是打开了，如果你是首先打开这个赛道的人，肯定会拿到这个赛道最大的红利。”**

腾讯视频用《三体》剧集打开科幻剧集赛道的故事，精彩程度不亚于《三体》本身。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7tMuAmiadsyvAk4l30TZvmgP03RGX0iaosuL5yVawsdblYqeWUcOTHYoQ/640?wx_fmt=png)**“选择开发《三体》，因为我自己对这个题材有偏爱”**

“这是一个由中国人主导的、影响全人类和宇宙的故事，在很多知名科幻作品中，是没有出现过中国名字的，而现在我们知道汪淼、史强，知道章北海、罗辑......这本身就意义重大”。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bk4v7Xre1fTVAlv8jvkR32brpibmV0dNlVRh03WRKSiaRubgoIU4qpc26w/640?wx_fmt=png)

在采访中，《三体》的意义被反复提及，和《流浪地球》一样，做出“说中国话的科幻作品”，几乎是中国所有文娱人的共同期望。

2015年，刘慈欣的《三体I：地球往事》斩获第73届雨果奖最佳长篇故事奖，这是中国作家首次拿下该奖项。在科幻圈内，《三体》的大名无人不知，而且呈现小范围出圈的态势，IP改编又是市场看好的热门方向，这部后来引发无数戏剧性情节的科幻小说，摆在了腾讯视频的项目会上。

读过原著的孙忠怀自然清楚《三体》的分量，不希望错过这个机会，然而这一想法在公司内部的决策会上遇到了困难。“决策会有二十多个人参加，大家意见分歧很大，反对的人很多，争论也很激烈。”

“不太甘心。”这是孙忠怀在决策会上面对反对意见的想法，作为一个重度科幻爱好者，《基地》系列、丹·西蒙斯的《海伯利安》、《深渊上的火》、《天渊》……这些科幻巨著他都如数家珍，当然也包括《三体》。“说实话在当时来看，这是一个比较感性的决策，也冒了很大风险，不太符合我平时相对理性的做事风格。”孙忠怀如是说。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bkxaYXeEniaXZtcfmwXjvqEqOGbQBfhsiaFsgJjzSHBCzUpsVdnX9PANBQ/640?wx_fmt=png)

如果回看七年前的电视剧市场，其逻辑和受众跟今天完全不一样，对于当时的中国市场来说，科幻题材无论电影还是电视剧都没有经过验证，更关乎整个国家的影视工业化水平，《流浪地球》2017年立项时，行业对其关注度和预期并不高。

所以孙忠怀坦言，“反对是对的，从各方面考量来看，当时的市场不太会做科幻题材，它很难驾驭，也相对太小众了。”

但在他看来，除了科幻本身的吸引力，《三体》与中国社会历史与现实的强联系，把它的社会价值提升到了新的高度。另一方面，《三体》的叙事尺度是宇宙级的，中国人在故事中是绝对主角，但又没有落入个人英雄主义拯救地球的俗套，即便剥离科幻的外壳，依然是一部值得玩味和思考的作品，而这正是它具备出圈潜力的原因。

从《开端》到《三体》，腾讯视频几乎是唯一下注科幻剧集赛道并持续投入其中的中国视频平台，实际上，采访时孙忠怀就向小娱透露了后续一系列科幻作品的开发规划，**拓新赛道的缘起可能是勇气，也可能是野心，但最终还是要回归经验和方法，才会有持续的势能和红利。**

《三体》成功播出后，对于科幻影视化的潜力标准，腾讯视频和孙忠怀也有了一套新的评估系统，第一优先级的考量就在于故事内核，其次是视觉呈现。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bkrxicTBSsb4ljIQ6F6WBMdBHibhbO2VYCiargrc8WG10PicVAiaPcic06w4WQ/640?wx_fmt=png)

“它的精神风貌、核心价值观是什么，是不是符合我们时代的价值观，在这个基础上再看人物、故事情节、桥段，有一些科幻作品太过于灰色，就不太适合”。而从落地的角度看，孙忠怀认为外在呈现更多的科幻作品更容易影视化，但在涉及到宇宙和外太空的场面时，制作难度又会指数级提升。

而已经被腾讯视频拿下，时间跨度大、场景多的《三体I：地球往事》和《三体II：黑暗森林》等作品，正是故事内核坚硬，外在呈现极难的作品。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7ejDSZf97dAE3mMYqSpwDp0blV0YsOONibSOjLz8EycRV8uxj7xc8QIg/640?wx_fmt=png)**《三体》开发难，难在何处？**

拍《三体》，到底目标受众是谁？

这是腾讯视频和制作团队最重要也最冒险的问题。

最初的考虑是走大众路线，对原著进行大幅度改编，做成一个罪案题材的作品，在罪案之外增加科幻元素。毕竟科幻小说可以自由地放飞想象，但是影视化必须考虑现实落地的可行性，这样不仅可以照顾更多的观众，还能节约成本，但孙忠怀觉得，“这么拍，过不了自己那关”。

几经商讨，直到2019年，团队最终敲定了《三体》回归原著的方向，拍原汁原味的《三体》电视剧。

同时在这段时间里，《三体》影响力的不断扩大，基于雨果奖的影响力，包括商界领袖和名人政要在内的更多人成为《三体》书粉，不遗余力地推介此书，为《三体》奠定了良好的观众基础。

更多的变化正是在这几年中发生的，超过一亿规模的腾讯视频会员基本盘，用户的付费习惯逐渐养成，商业逻辑也有变化，《三体》剧集在优质的平台、恰当的时间遇到了足够多核心受众，这都印证了孙忠怀所说的“一剧一命”。

在合作伙伴的选择上，腾讯视频经过慎重考量，最终选择了与灵河文化一起开发剧版《三体》，2014年，由白一骢深度参与创作的《暗黑者》在孙忠怀看来就是“开赛道”之作，而到了《三体》这个再开赛道的机会，腾讯视频和白一骢团队凭借信任再次走到了一起。

就导演人选而言，用户对杨磊导演此前的认知以年代大戏《红色》为代表，但孙忠怀告诉小娱，市面上鲜有特效班底固定的剧集导演，但杨磊和特效导演陆贝珂是常年固定合作的搭档，彼此很有默契，对于《三体》高投入大制作的作品，稳定性和团队协调性很重要，而且两人作为《三体》的书迷，对于作品有发自真心的热爱，这也是确定人选的重要因素。

科幻剧不像常规电视剧，往往涉及大量科学原理、科学理论的相关知识，因此前期的筹备工作需要花费大量的精力，光这部分工作就耗时一两年。开拍过程中同样也难度重重，CG、动作捕捉、实景拍摄……每一个困难都耗费了团队大量的心血。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bkoGwJiaEXFq2826BiaEVdgtr9XKCp7vDOFcxeSVfUW8fkicwia6KfEfmI3A/640?wx_fmt=png)

杀青之后的后期工作也不轻松，据杨磊透露，“整体剪辑大概花了一年半时间。”和一般的电视剧剪辑不同，《三体》有大量的戏需要靠感受、情绪来推进，那么如何选择恰当的方案就成了关键，只能反复比对，不停尝试，以至于在最初的三个月里，只剪好了一集。

如此精雕细琢的创作态度，必然会导致成本的上升，但是像《三体》这样“定海神针”量级的作品，不能只看商业价值，还要注重艺术价值和口碑价值，这些东西是不能用钱来衡量。

基于这些考虑，腾讯视频对于《三体》开了不少绿灯。像红岸基地的戏份，室内戏是在北京进行场地搭景拍摄，但想拍全景只能在冬天拍摄，最终剧组选择了先解散，冬天再集合的策略，在冬天前往黑龙江的黑河拍摄，最终达到了想要的效果。

最终，《三体》开播后的口碑没有辜负这些付出，不光赢得了书粉的赞扬，也吸引了大批路人观众，扩大了中国科幻的基本盘。这些成绩都说明，《三体》找到了一种适合国内科幻的解法，那就是紧扣原著。

以往的 IP 改编或多或少都会有对原著的解构，但现在看来，对于科幻作品来说，尽可能忠实原著基本不会错。回到《三体》本身，它跟社会的联结、所具有的企业洞察以及和现实生活的深刻关联，都让它更接地气，也更容易被普通观众接受。

在娱乐资本论看来，立足原著，服务书粉，辐射路人，以科幻作品的基本盘撬动破圈的实现，是一条国内科幻改编可以复制的道路。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8r0eeusveAtyj98pKeBEz7gMbSIRF8ujdpJibC3CLgiaEEY6kJq4YuKUC4cv1ZG4kjEVEHhs35Zn3Q/640?wx_fmt=png)**《三体》是腾讯视频打开科幻赛道的钥匙**

毋庸置疑，《三体》之于中国科幻，是一部里程碑式的作品。长期以来，科幻在中国影视版图里始终地位不显，如果说《流浪地球》让国产科幻电影有底气在电影领域立足，那么《三体》就让国产科幻剧在电视剧领域扎下了深根。

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bk32LzAicricJULWw6ibib1BiahCfTNqc72cj2EI0zYqWwsUul7GxUnxJ5z8w/640?wx_fmt=png)

好的作品可以打开一个赛道，《开端》成功的吸引了一批对科幻没那么感兴趣的观众，而《三体》的影响力更是出圈，也确定了科幻赛道被打开。孙忠怀对此表示：

“首先打开这个赛道的人，肯定会拿到这个赛道最大的红利。科幻赛道会从很难、很小，变得没有那么难、那么小，当然可能无法像常规电视剧那么大众化，但也比我们原来预想的要好很多。”

从之前《开端》到如今的《三体》，腾讯在科幻赛道上持续耕耘，终于有所收获。《三体》成功拓展了对科幻感兴趣的基本盘，有不少书迷疯狂的向身边人安利，带动了大量观众被《三体》吸引，形成破圈势能。

但孙忠怀也坦言，像《三体》这样规模宏大、质量上乘的作品并不好找，未来腾讯视频可能也会针对一些小体量的科幻作品进行改编，以推出短剧集的形式来巩固科幻赛道，而未来一段时间的主要精力，肯定是投入到《三体2》的制作中去。

回望《三体》项目的这七年，所有的付出都是值得的。得益于前期的努力，让《三体》的口碑立得住，也让腾讯视频对《三体II：黑暗森林》有了信心。据孙忠怀透露，目前续作已经处于大纲阶段。因为将会涉及到太空画面的制作，比起前作难度更大，但是第一部已经给予了他们足够多的信心，也明确了第二部的改编方向，压力依然存在，但孙忠怀却并不担忧，**“当你在三四月做了该做的事，七八月就一定会有收获。”**

![](https://mmbiz.qpic.cn/mmbiz_png/jNZszpkibXx8RsHhB5uLzneoRnAia889bkNw6XGgLepib3ahzpBSibNSjL0oj9OmdiaBEgHpD5uIyfCAdRicsBqdpYZQ/640?wx_fmt=png)

在科幻赛道上，腾讯视频已经摸索出了一条前行之路，有些项目很难，但是值得全力以赴，正是因为这样的全力以赴，才让中国科幻的土壤逐渐走向成熟。日前，国家广播电视总局发展研究中心发表评论文章，高度评价《三体》剧集的价值：

*“电视剧《三体》创作团队以7年的时间精心构思、耐心打磨作品，为观众奉献了一部写意与写实并重、现实与科幻交融的中国气派的科幻剧，为国际市场贡献了一部别具一格的中国式科幻剧，为海外观众提供了新选项。”*

对于腾讯视频和整个在线视频行业来说，《三体》剧集成功的意义远远超过剧集赛道本身。在这部作品之前，国内观众很难在脑海中将“科幻”和“国产剧”挂钩，但《三体》剧集本身过硬的质量，将具备超越时间的力量，也让国内外的观众意识到，影响全人类和宇宙的名字，可以是叶文洁、史强、汪淼、罗辑、章北海，正如孙忠怀所言：

**“希望将来中国观众不只去看海外的作品，也能对中国的作品有所期待，随着创作的提升、故事的丰富，将来想看科幻作品的时候，中国作品也一定会是首选之一。”**
