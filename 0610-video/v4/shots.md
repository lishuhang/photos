## 动效类型

| 动效名 | CSS实现 | 应用场景 |
|--------|---------|----------|
| fadeIn | opacity 0→1 | 通用入场 |
| slideUp | translateY(40px→0) + opacity | 卡片、文字 |
| slideLeft | translateX(-40px→0) + opacity | 条形图条目 |
| slideRight | translateX(40px→0) + opacity | 右侧数据 |
| popIn | scale(0→1.1→1) + opacity | 数字、标签 |
| flashIn | opacity快速闪烁→1 | 标题 |
| zoomIn | scale(0.5→1) + opacity | 结尾CTA |
| barGrow | scaleX(0→1) | 横向条形图 |
| barGrowUp | scaleY(0→1) | 纵向柱状图 |
| radarFadeIn | scale(0.3→1) + opacity | 雷达图 |
| bubblePop | scale(0→1.1→1) | 气泡图 |
