## 动效类型速查

| 代号 | 效果 | CSS实现 |
|------|------|---------|
| fadeIn | 透明→不透明 | opacity 0→1 |
| fadeOut | 不透明→透明 | opacity 1→0 |
| slideUp | 从下方滑入 | translateY(50px)→0 + opacity |
| slideLeft | 从左滑入 | translateX(-50px)→0 + opacity |
| slideRight | 从右滑入 | translateX(50px)→0 + opacity |
| popIn | 弹性缩放 | scale(0.5)→1.05→1 + opacity |
| flashIn | 白闪后稳定 | brightness(3)→1 + opacity |
| zoomIn | 缩放进入 | scale(1.3)→1 + opacity |
| shrinkOut | 缩小退出 | scale(1)→0.5 + opacity |
| typewriter | 逐字出现 | JS控制字符逐步显示 |
| cascadeIn | 级联依次入场 | 多元素stagger delay + slideUp |
| bubblePop | 气泡弹出 | scale(0)→1.1→1 + opacity |
