## 制作方法

### HTML动画架构
- 单一HTML文件，所有CSS/JS内联
- JavaScript timeline控制器：requestAnimationFrame循环，精确到毫秒级
- 每个场景div有data-start和data-dur属性，JS根据时间轴切换active状态
- 图表动画复用skill06-v2的Pure CSS动画代码
- SRT字幕数据JSON化嵌入JS，按时间轴显示/隐藏
- 音频通过<audio>标签播放，与动画同步启动

### 素材来源
- 12家公司Logo：官网/Wikipedia/萌娘百科下载
- AI生成素材：z-ai-generate CLI工具生成4张背景/概念图
- Word文档图片：python-docx提取9张原文配图
- 字体：阿里巴巴普惠体5种字重woff文件

### 录屏方法
- Playwright headless Chromium内置video recording
- `context.new_context(record_video_dir=..., record_video_size=...)`
- 等待145秒动画播放完毕后关闭context生成WebM
- ffmpeg转码为MP4 + 添加AAC音频轨道

### 关键技术细节
1. **daemon进程模式**：sandbox超时限制2分钟，用double-fork守护进程确保145秒录制不被中断
2. **字幕渲染**：HTML内绝对定位，阿里巴巴普惠体，text-shadow四方向黑色描边
3. **图表动画触发**：CSS animation-delay在场景激活时才开始计时
4. **音频同步**：点击body触发audio.play()，与animation同时启动
